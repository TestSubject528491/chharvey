Predicate Logic

A set is that which may own sets.

Given sets \(s\) and \(t\), the assertion that \(s\) owns \(t\) is a proposition, denoted \(t\in s\). We may also say that \(t\) is an element of, or a member of, \(s\).

We use the predicate \(t\not\in s\) to refer to the negation \(\lnot(t\in s)\).

To avoid ambiguity, we will not use the word contain to represent ownership. For historical reasons, this expression can be interpreted as an indication of ownership or one of inclusion.

The Principle of Bivalence, Predicate Version Given sets \(t\) and \(s\):

  • The Law of Excluded Middle It is always the case that \(t\) is an element of \(s\) or not an element of \(s\).

    \[(t\in s) \lor \lnot(t\in s)\]

  • The Law of Non-Contradiction It is never the case that \(t\) is both an element of \(s\) and not an element of \(s\).

    \[\lnot\bigl((t\in s) \land \lnot(t\in s)\bigr)\]

use Principle of Bivalence, Proposition Version

Given a set \(x\), the predicate \(\l P_x\) is a proposition that makes an assertion involving \(x\).

In the natural numbers, let \(\l P_n\) be the predicate \(n\) is even. Then \(\l P_3\) would be false, while \(\l P_6\) would be true.

Given a predicate \(\l P_x\), the assertion that there exists a set \(x\) that makes \(\l P_x\) true is itself a predicate, called an existential quantifier.

\[(\exists x)(\l P_x)\]

We will sometimes use the predicate \((\not\exists x)(\l P_x)\) to refer to the negation \(\lnot\bigl((\exists x)(\l P_x)\bigr)\), however we will see in D-forall that this notation—and even the predicate itself—is not necessary; the concept of negating an existential quantifier is encapsulated in another kind of predicate, called a universal quantifier.

The motivation for such an object arises from this: if there is an assertion that there exists a set that forces some predicate to be true, then the negation of that assertion is equivalent to the assertion that that predicate is automatically false for every set.

The same is true for the value of the predicate: If we negate the assertion that there exists a set that forces the predicate to be false, then we are asserting that the predicate must be true for every set.

Given a predicate \(\l P_x\), the assertion that there does not exist any set \(x\) that makes \(\l P_x\) false is itself a predicate, \((\not\exists x)(\lnot\l P_x)\), called a universal quantifier. This is the assertion that \(\l P_x\) is true for any set \(x\).

\[(\forall x)(\l P_x)\]

Given a predicate \(\l P_x\) and a set \(y\), the assertion that there exists an element \(x\) of \(y\) that makes \(\l P_x\) true is a predicate called a bound existential quantifier.

\[ \left((\exists x\in y)(\l P_x)\right) \equiv \left((\exists x)((x\in y) \land \l P_x)\right) \]

let y be a set with members a, b, c, ... z. Then
\[ \left((\exists x\in y)(\l P_x)\right) \iff \left( \l P_a \lor \l P_b \lor \l P_c \lor \dots \lor \l P_z \right) \]

Given a predicate \(\l P_x\) and a set \(y\), the assertion that \(\l P_x\) is true for all elements \(x\) of \(y\) is a predicate called a bound universal quantifier.

\[ (\forall x\in y)(\l P_x) \equiv (\forall x)((x\in y) \implies \l P_x) \]

\[ \bigl((\forall x\in y)(\l P_x)\bigr) \iff \bigl((\not\exists x\in y)(\lnot\l P_x)\bigr) \]

\[ (\forall x\in y)(\l P_x)\\ (\forall x)(x\in y \implies \l P_x)\\ (\forall x)(x\not\in y \lor \l P_x)\\ (\forall x)(\lnot(x\in y \land \lnot\l P_x))\\ (\not\exists x)(x\in y \land \lnot\l P_x)\\ (\not\exists x\in y)(\lnot\l P_x)\\ \]

let y be a set with members a, b, c, ... z. Then
\[ \left((\forall x\in y)(\l P_x)\right) \iff \left( \l P_a \land \l P_b \land \l P_c \land \dots \land \l P_z \right) \]

Combined Quantifiers

Given an existential quantifier \((\exists y)(\l P_{x,y})\), the existential quantifier \(\bigl(\exists x\bigr)\bigl((\exists y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\exists x)(\exists y)(\l P_{x,y})\]

Given a universal quantifier \((\forall y)(\l P_{x,y})\), the universal quantifier \(\bigl(\forall x\bigr)\bigl((\forall y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\forall x)(\forall y)(\l P_{x,y})\]

existential quantification is commutative

\[ \bigl((\exists x)(\exists y)(\l P_{x,y})\bigr) \equiv \bigl((\exists y)(\exists x)(\l P_{x,y})\bigr) \]

universal quantification is commutative

\[ \bigl((\forall x)(\forall y)(\l P_{x,y})\bigr) \equiv \bigl((\forall y)(\forall x)(\l P_{x,y})\bigr) \]

an existential disjunction is logically equivalent to a disjunction of existential quantifiers

\[ \bigl((\exists x)(\l P_x \lor \l Q_x)\bigr) \equiv \bigl((\exists a)(\l P_a) \lor (\exists b)(\l Q_b)\bigr) \]

a universal disjunction is logically equivalent to a disjunction of universal quantifiers

\[ \bigl((\forall x)(\l P_x \lor \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \lor (\forall b)(\l Q_b)\bigr) \]

an existential conjunction logically implies a conjunction of existential quantifiers

\[ \bigl((\exists x)(\l P_x \land \l Q_x)\bigr) \therefore \bigl((\exists a)(\l P_a) \land (\exists b)(\l Q_b)\bigr) \]

a universal conjunction is logically equivalent to a conjunction of universal quantifiers

\[ \bigl((\forall x)(\l P_x \land \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \land (\forall b)(\l Q_b)\bigr) \]

an existential implication logically implies an implication of existential quantifiers

\[ \bigl((\exists x)(\l P_x \implies \l Q_x)\bigr) \therefore \bigl((\exists a)(\l P_a) \implies (\exists b)(\l Q_b)\bigr) \]

a universal implication is logically equivalent to an implication of universal quantifiers

\[ \bigl((\forall x)(\l P_x \implies \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \implies (\forall b)(\l Q_b)\bigr) \]

A universal implication is logically equivalent to the contrapositive of existential quantifiers

\[ \bigl((\forall x)(\l P_x \implies \l Q_x)\bigr) \equiv \bigl((\exists a)(\lnot\l Q_a) \implies (\exists b)(\lnot\l P_b)\bigr) \]

an implication of existential quantifiers is logically equivalent to a universal implication whose consequent is an existential quantifier

\[ \bigl((\exists a)(\l P_a) \implies (\exists b)(\l Q_b)\bigr) \equiv \Bigl(\bigl(\forall x\bigr)\bigl(\l P_x \implies (\exists y)(\l Q_{x,y})\bigr)\Bigr) \]

specification?

\[ \Bigl(\bigl(\exists x\bigr)\bigl((\forall y)(\l P_{x,y})\bigr)\Bigr) \therefore \Bigl(\bigl(\forall y\bigr)\bigl((\exists x)(\l P_{x,y})\bigr)\Bigr) \]

Equality and Inclusion

The Intensionality Definition A set is equal to another set iff those sets own each other's elements.

We use the predicate \(x=y\) to describe the assertion that a set \(x\) is equal to a set \(y\). We define this predicate as:

\[ \left(x=y\right) \equiv \left((\forall t)(t\in x \iff t\in y)\right) \]

We use the predicate \(x\not= y\) to refer to the negation \(\lnot(x=y)\).

The Axiom of Extensionality Equal elements are members of the same sets.

\[ (\forall x)(\forall y) \left( \expand{\(x=y\)}{ (\forall t)(t\in x\iff t\in y) } \iff \left((\forall z)(x\in z \iff y\in z)\right) \right) \]

Intensionality states that sets that are equal have exactly the same members. We take this to be the definition of equality: equal sets are identical. One very important interpretation of this is that a set is determined uniquely its elements, so if sets with exactly the same elements have different definitions, names, or rearrangements, those sets are all one in the same. This may seem obvious, but a more fascinating fact about equal sets follows.

By The Axiom of Extensionality, all sets are definite and distinguishable. This means that given a set, one should be able to decide whether or not any particular element would belong to that set; and that any two distinct elements of the same set are unequal. This will come in handy later on, when we start repeating elements within a set. By this axiom, a set cannot own the same element more than once.

A set \(x\) is a subset of a set \(y\) iff \(y\) owns all the elements of \(x\).

We use the predicate \(x\subseteq y\) to describe the assertion that \(x\) is a subset of \(y\). We define this predicate as:

\[ \left(x\subseteq y\right) \equiv \expand{\((\forall t\in x)(t\in y)\)}{(\forall t)(t\in x \implies t\in y)} \]

We also say \(x\) is included in \(y\), or \(y\) includes \(x\).

A set \(x\) is a proper subset of a set \(y\) iff \(x\subseteq y\) and \(x\not= y\).

We use the predicate \(x\subset y\) to describe the assertion that \(x\) is a proper subset of \(y\). We define this predicate as:

\[ \bigl(x\subset y\bigr) \equiv \bigl( (x\subseteq y) \land (x\not=y) \bigr) \]

We also say \(x\) is strictly included in \(y\), or \(y\) strictly includes \(x\).

Remember the distinction between \(y\) owns \(x\) and \(y\) includes \(x\).

In the former expression, we are trying to determine if \(x\) is an element of \(y\). To do so, we look at the set \(x\) as a single object, and then check to see if this object is among the members of the set \(y\). If so, \(x\in y\).

In the latter expression, we are trying to determine if \(x\) is a subset of \(y\). To do this, we must open up the set \(x\), examine its various elements, and then check to see if the set \(y\) owns every one of those elements. If so, \(x\subseteq y\).

In each case, we will avoid the term contain because of its ambiguous meaning in different sources. (A set could contain its elements or it could contain its subsets.)

Uniqueness Quantifiers

A uniqueness quantifier is an assertion that there exists a particular set, and that is the only one that exists. Essentially, by stating a uniqueness quantifier, we assert that there exists a set that exhibits a certain characteristic, and all sets that exhibit that characteristic are equal.

define: existence in the first statement; uniqueness in the second

There exists an x such that P_x is true, and for all x such that P_x is true, P_y is true if and only if x=y.

\[ \biggl((\exists!x)(\l P_x)\biggr) \equiv \biggl( \Bigl((\exists x)(\l P_x)\Bigr) \land \Bigl( \bigl(\forall x\bigr) \bigl(\l P_x \implies (\forall y)(\l P_y \iff x=y)\bigr) \Bigr) \Biggr) \]

Prove: There exists an x such that P_x is true, and for any two x and y, if P_x and P_y are true, then x=y.

\[ \biggl((\exists!x)(\l P_x)\biggr) \equiv \biggl( \Bigl((\exists x)(\l P_x)\Bigr) \land \Bigl( \bigl(\forall x\bigr)\bigl(\forall y\bigr) \bigl( (\l P_x \land \l P_y) \implies (x=y) \bigr) \Bigr) \biggr) \]

prove: existence and uniqueness in one statement

\[ \bigl((\exists!x)(\l P_x)\bigr) \equiv \Bigl( \bigl(\exists x\bigr) \bigl(\l P_x \land (\forall y)(\l P_y \iff x=y)\bigr) \Bigr) \]

The Axioms of Set Theory

The Seven Pillars of Creation

The seven axioms below serve as the foundation of all mathematics. The first six tell us what sets may be brought into existence, and the seventh specifies a important characteristic of sets that do exist.

Existence

A set is called empty iff it has no members.

We use the predicate \(s\doteqdot\varnothing\) to refer to the assertion that the set \(s\) is empty, and \(s\not\doteqdot\varnothing\) for the negation.

\[ \left(s\doteqdot\varnothing\right) \equiv \left((\forall t)(t\not\in s)\right) \]

\[ \left(t\in\varnothing\right) \equiv \bot \]

The Axiom of Existence There exists an empty set.

\[ \expand{\(\exists\varnothing\)}{ (\exists s)\expand{\(s\doteqdot\varnothing\)}{ (\forall t)\expand{\(t\not\in s\)}{\lnot(t\in s)} } } \]

To describe any set \(x\) as empty, we assert that \((\forall t)(t\not\in x)\) and use the notation \(x\doteqdot\varnothing\) as shorthand. But we now see that the notation has a quite literal meaning. That is, any set \(x\) that is empty is literally equal to the empty set \(\varnothing\). We can verify this with The Intensionality Definition. This goes to show that all empty sets are equal, as proven above.

In our informal discussions, we can reconcile this idea by replacing the shorthand notation with equality.

\[x\doteqdot\varnothing \equiv x=\varnothing\]

However in rigorous well-formed formulae we cannot use \(x=\varnothing\) because \(\varnothing\) is not technically a symbol in predicate logic.

Pair

A set is called a pair of two given sets iff it owns exactly those given sets.

We use the predicate \(s\doteqdot\{x,y\}\) to refer to the assertion that the set \(s\) is a pair of the sets \(x\) and \(y\), and \(s\not\doteqdot\{x,y\}\) for the negation.

\[ \left(z\in \{x,y\}\right) \equiv \left( (z=x) \lor (z=y) \right) \]

The Axiom of Pair For every two sets, there exists a set that owns exactly them both.

\[ (\forall x)(\forall y)\expand{\(\exists\left\{x,y\right\}\)}{ (\exists s)\expand{\(s\doteqdot\left\{x,y\right\}\)}{ (\forall z)\left( (z\in s) \iff \left( \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } \right) \right) } } \]

Power

A set is called a power set of a given set iff it owns exactly all subsets of the given set.

We use the predicate \(s\doteqdot\wp x\) to refer to the assertion that the set \(s\) is a power set of the set \(x\), and \(s\not\doteqdot\wp x\) for the negation.

\[ \left(y\in \wp x\right) \equiv \left(y\subseteq x\right) \]

The Axiom of Power There exists a set that owns exactly all subsets of any given set.

\[ (\forall x)\expand{\(\exists \wp x\)}{ (\exists s)\expand{\(s\doteqdot\wp x\)}{ (\forall y)\left( (y\in s) \iff \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } \right) } } \]

Union

A set is called a union of a given set iff it owns exactly any members belonging to any elements of the given set.

We use the predicate \(s\doteqdot\bigcup x\) to refer to the assertion that the set \(s\) is a union of the set \(x\), and \(s\not\doteqdot\bigcup x\) for the negation.

\[ \left(y\in\bigcup x\right) \equiv \left( (\exists z\in x)(y\in z) \right) \]

Given the pair \(\left\{u,v\right\}\), the union \(\bigcup\left\{u,v\right\}\) is called, for simplicity, the union of \(u\) and \(v\) and is denoted \(u\cup v\). It owns exactly any members belonging to \(u\) or \(v\).

The Axiom of Union There exists a set that owns exactly any element belonging to any element of a given nonempty set.

\[ (\forall x)\left( \expand{\(x\not\doteqdot\varnothing\)}{ (\exists t)(t\in x) } \implies \expand{\(\exists\bigcup x\)}{ (\exists s)\expand{\(s\doteqdot\bigcup x\)}{ (\forall y)\left( (y\in s) \iff \expand{\((\exists z\in x)(y\in z)\)}{ (\exists z)\left( (z\in x) \land (y\in z) \right) } \right) } } \right) \]

Intersection

A set is called a union of a given set iff it owns exactly all members belonging to all elements of the given set.

We use the predicate \(s\doteqdot\bigcap x\) to refer to the assertion that the set \(s\) is an intersection of the set \(x\), and \(s\not\doteqdot\bigcap x\) for the negation.

\[ \left(y\in\bigcap x\right) \equiv \left( (\forall z\in x)(y\in z) \right) \]

Given the pair \(\left\{u,v\right\}\), the intersection \(\bigcap\left\{u,v\right\}\) is called, for simplicity, the intersection of \(u\) and \(v\) and is denoted \(u\cap v\). It owns exactly all members belonging to \(u\) and \(v\).

The Axiom of Intersection There exists a set that owns exactly all elements belonging to all elements of a given nonempty set.

\[ (\forall x)\left( \expand{\(x\not\doteqdot\varnothing\)}{ (\exists t)(t\in x) } \implies \expand{\(\exists\bigcap x\)}{ (\exists s)\expand{\(s\doteqdot\bigcap x\)}{ (\forall y)\left( (y\in s) \iff \expand{\((\forall z\in x)(y\in z)\)}{ (\forall z)\left( (z\in x) \implies( y\in z) \right) } \right) } } \right) \]

Infinity

A set is called inductive iff it is nonempty and each of its elements is owned by another element of the set.

The Axiom of Infinity There exists a nonempty set of which each element is owned by another element of this set.

\[ (\exists s)\expand{\(s\) is inductive}{ \expand{\(s\not\doteqdot\varnothing\)}{ (\exists t)(t\in s) } \land \expand{each element of \(s\) is owned by of another element of \(s\)}{ (\forall x)\left( (x\in s) \implies \expand{\(x\) is a member of another element of \(s\)}{ (\exists y)\left( (y\in s) \land (x\in y) \right) } \right) } } \]

At this point, we do not have a formal definition for a set that is infinite, but informally we can think of infinite sets as having never-ending elements: there are more elements than we can count. One way to achieve this is to assert the existence of an inductive set. Indeed, it is true that all inductive sets are infinite. The converse is left as a cliffhanger.

The intent of The Axiom of Infinity is that there exists an infinite set; that is, a set with an unlimited number of elements. However, D-inductive only requires that an inductive set be nonempty, and own at least one element for every element it owns.

For example, it is not unreasonable to consider the sets \(a\) and \(b\) such that \(a\in b\) and \(b\in a\). Formally, we are not guaranteed the existence of such sets, but let's hypothesize that they do exist. Certainly, the pair \(\{a,b\}\) exists, and indeed it does meet the requirements of D-inductive: \(\{a,b\}\) is nonempty, and for every \(x\in \{a,b\}\), there exists a \(y\in \{a,b\}\) such that \(x\in y\).

It is to our disappointment that a set such as the one above is inductive but is not infinite (satisfying our intuitive understanding of infinite sets). Thus we must create a new rule that does not allow sets to own themselves, or to own elements that own themselves, or so on. Essentially, we cannot allow the chain of membership to be cyclical. This will ensure that for any element in an inductive set, we must have a new element—one that hasn't been mentioned before—in the set. This will satisfy our intuitive notion of a set with more elements than we can count.

Regularity

Below is the seventh axiom of the foundation of mathematics. Rather than declaring the existence of certain sets, as the first six do, this axiom merely states a fact about sets that already exist.

A set is called regular iff it is empty or it has an element that does not own another element of the set.

The Axiom of Regularity Every nonempty set has an element that does not own another element of the set.

\[ (\forall s)\expand{\(s\) is regular}{ \expand{\(s=\varnothing\)}{ (\forall t)\expand{\(t\not\in s\)}{\lnot(t\in s)} } \lor \expand{there exists an element of \(s\) that does not own another element of \(s\)}{ (\exists x)\left( (x\in s) \land \expand{no members of \(x\) are elements of \(s\)}{ (\forall y)\left( (y\in s) \implies \expand{\(y\not\in x\)}{\lnot(y\in x)} \right) } \right) } } \]

Consequences of the Axioms

Uniqueness

The Axioms above merely state the existence of certain sets. Some of those sets have uniqueness, meaning that any two sets that are constructed the same way are equal. This section proves or disproves uniqueness for each axiom.

The empty set is unique.

\[ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\varnothing\)}{ (\forall t)\expand{\(t\not\in a\)}{\lnot(t\in a)} } \land \expand{\(b\doteqdot\varnothing\)}{ (\forall t)\expand{\(t\not\in b\)}{\lnot(t\in b)} } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) \]

A pair constructed from two given sets is unique.

\[ (\forall x)(\forall y)\expand{\(\left\{x,y\right\}\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\left\{x,y\right\}\)}{ (\forall z)\left( (z\in a) \iff \left( \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } \right) \right) } \land \expand{\(b\doteqdot\left\{x,y\right\}\)}{ (\forall z)\left( (z\in b) \iff \left( \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } \right) \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

The power set constructed from a given set is unique.

\[ (\forall x)\expand{\(\wp x\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\wp x\)}{ (\forall y)\left( (y\in a) \iff \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } \right) } \land \expand{\(b\doteqdot\wp x\)}{ (\forall y)\left( (y\in b) \iff \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

The union constructed from a set is unique.

\[ (\forall x)\expand{\(\bigcup x\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\bigcup x\)}{ (\forall y)\left( (y\in a) \iff \expand{(\exists z\in x)(y\in z)}{ (\exists z)\left( (z\in x) \land (y\in z) \right) } \right) } \land \expand{\(b\doteqdot\bigcup x\)}{ (\forall y)\left( (y\in b) \iff \expand{(\exists z\in x)(y\in z)}{ (\exists z)\left( (z\in x) \land (y\in z) \right) } \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

(\forall u)(\forall v)(\forall x)(\forall z)(z\in x \iff (z=u)\lor(z=v))(\forall a)(\forall b)((((\forall y)(y\in a \iff (\exists z)(z\in x\land y\in z)))\land((\forall y)(y\in b \iff (\exists z)(z\in x\land y\in z))))\implies a=b)

The intersection constructed from a set is unique.

\[ (\forall x)\expand{\(\bigcap x\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\bigcap x\)}{ (\forall y)\left( (y\in a) \iff \expand{(\forall z\in x)(y\in z)}{ (\forall z)\left( (z\in x) \implies (y\in z) \right) } \right) } \land \expand{\(b\doteqdot\bigcap x\)}{ (\forall y)\left( (y\in b) \iff \expand{(\forall z\in x)(y\in z)}{ (\forall z)\left( (z\in x) \implies (y\in z) \right) } \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

Injectiveness

Injectiveness is the opposite of uniqueness. While uniqueness states that any two sets that are constructed the same way are equal, injectiveness states that if two equal sets are constructed from given sets, those given sets are equal. This section proves or disproves injectiveness for each axiom.

If two power sets are equal, their base sets are equal.

Singletons

Given a set \(x\), a set is called a singleton of \(x\), denoted \(\left\{x\right\}\), iff it owns exactly \(x\).

\[ \left(y\in \{x\}\right) \equiv \left(y=x\right) \]

We use the predicate \(s\doteqdot\left\{x\right\}\) to refer to the assertion that a set \(s\) is a singleton of \(x\).

Every set has a singleton.

\[ (\forall x)\expand{\(\exists\left\{x\right\}\)}{ (\exists s)\expand{\(s\doteqdot\left\{x\right\}\)}{ (\forall y)\left( (y\in s) \iff \expand{\(y=x\)}{ (\forall t)(t\in y \iff t\in x) } \right) } } \]

A singleton constructed from one set is unique.

\[ (\forall x)\expand{\(\{x\}\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteqdot\{x\}\)}{ (\forall y)\left( (y\in a) \iff \expand{\(y=x\)}{ (\forall t)(t\in y \iff t\in x) } \right) } \land \expand{\(b\doteqdot\{x\}\)}{ (\forall y)\left( (y\in b) \iff \expand{\(y=x\)}{ (\forall t)(t\in y \iff t\in x) } \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

This theorem shows the singleton function is well-defined: if two sets are equal, their singletons must be equal.

The element of a singleton is unique.

This theorem shows the singleton function is injective: if two singletons are equal, their elements must be equal.

Consequences of Regularity

Membership is antireflexive.

\[ (\forall x)\expand{\(x\not\in x\)}{\lnot(x\in x)} \]

Direct proof: Let \(x\) be any set. Then by T-singletonexists, \(\exists\{x\}\). We will apply A-regularity to \(\{x\}\): since \(\{x\}\not=\varnothing\), there must be an element of \(\{x\}\) that does not own any elements of \(\{x\}\). Since the only element of \(\{x\}\) is \(x\), then \(x\) itself must not own any elements of \(\{x\}\), which, namely, is \(x\). Therefore, \(x\not\in x\).

Alternate proof: Proof by contradiction: Suppose to the contrary that \((\exists x)(x\in x)\). By T-singletonexists, there exists a set \(\{x\}\). Since \(\{x\}\) is a singleton, its only element is \(x\), and by our assumption, \(x\in x\). Therefore all elements of \(\{x\}\) own an element of \(\{x\}\). This contradicts A-regularity.

Membership is antisymmetric.

\[ (\forall x,y)\left( (x\in y) \implies \expand{\(y\not\in x\)}{\lnot(y\in x)} \right) \]

Proof by contradiction: Given sets x and y, assume x∈y and y∈ x. Now consider the pair {x,y} (which exists and is unique). A-regularity says if {x,y} is nonempty, then ∃ z ∈ {x,y} such that ∀ w ∈ z, w∉ {x,y}. If z∈{x,y}, there are two cases:

Case 1: z=x. y∈z and y∈{x,y}

Case 2: z=y. x∈z and x∈{x,y}

Either way, if z∈{x,y}, then ∃w∈z such that w∈{x,y}. Therefore our assumption (x∈y and y∈x) is faulty; the negation must be true; namely (x∈y implies y∉x).

There is no set of which every set is a member.

\[ (\forall x)(\exists y)\expand{\(y\not\in x\)}{\lnot(y\in x)} \]

Assume to the contrary that \((\exists x)(\forall y)(y\in x)\). Then it must be the case that \(x\in x\). By T-membershipantireflexive, we have a contradiction.
