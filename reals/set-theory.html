Section II.A: Predicate Logic

A set is that which may own elements. These elements, in turn, are also sets.

There are a number of other ways to express set ownership.
\[t\in S\] \[S\owns t\]
\(t\) is an element of \(S\) \(S\) owns \(t\)
\(t\) is a member of \(S\) \(S\) is a possessor of \(t\)

If \(S\) does not own \(t\), we state, \(\lnot(x\in X)\), \(x\not\in X\), or \(X\not\owns x\).

To avoid ambiguity, we will not use the word contain to represent ownership. For historical reasons, this expression can be interpreted as an indication of ownership or one of inclusion.

Given a set \(t\), the predicate \(\l P_t\) is a proposition that makes a statement about \(t\).

In the natural numbers, let \(\l P_n\) be the proposition, \(n\) is even. Then \(\l P_3\) would be false, while \(\l P_6\) would be true.

Let \(\l p\) be a proposition, and let \(P\) and \(t\) be sets. Then

\[\l p_t \iff t\in S\]

example: in the natural numbers, let P_x be represented by the proposition, x is even. Then P_3 would be false, while P_6 would be true.

Definition: existential quantifier

Given a set S and elements a, b, c, d,..., and so on, of S, and given a predicate P_x, the existential quantifier (\exists x)(P_x) is defined by the predicate \l P_a \lor \l P_b \lor \l P_c \lor ..., and so on

Definition Given a proposition \(\l p\), the assertion that there exists an object \(x\) such that \(\l p\) holds is itself proposition, denoted

\[\exists x : (\l p)\]

The symbol \(\exists\) is called the existential quantifier.

Definition: universal quantifier

Given a predicate \l P_x, the universal quantifier (\forall x)(\l P_x) is defined by the negation

\lnot((\exists x)(\lnot\l P_x)) sometimes denoted (\notexists x)(\lnot\l P_x)

Definition 

		definitions: bound quantifiers
		
			(\exists x\in S)(P_x) is defined by (\exists x)(x\in S \land P_x)
			
			(\forall x\in S)(P_x) is defined by (\forall x)(x\in S \implies P_x)
		
		
			Definition: uniqueness quantifier
			(\exists!x)(P_x) defined by
			(\exists x)(P_x \land (\forall y)(P_y \iff x=y))
		
			combined quantifiers
			
			prove:
			(\exists x)((\forall y)(P_x,y)) \therefore (\forall y)((\exists x)(P_x,y))

Section II.B: The Axioms of Set Theory

A set \(A\) is equal to to a set \(B\) if and only if \(A\) and \(B\) contain each other's elements. That is,

\[(A=B) \text{ if and only if } (\forall t)(t\in A \iff t\in B)\]

This definition is often called The Axiom of Extensionality.

The Principle of Bivalence, Set Theory Version

Given sets \(t\) and \(S\):

  • The Law of Excluded Middle It is always the case that \(t\) is an element of or not an element of \(S\).

    \[(t\in S) \lor (t\not\in S)\]

  • The Law of Non-Contradiction It is never the case that \(t\) is both an element of and not an element of \(S\).

    \[\lnot((t\in S) \land (t\not\in S))\]

The Axiom of Existence There exists an empty set, and that set is unique.

\[\expand{There exists an empty set, and that set is unique.}{ (\exists x)\expand{\(x\) is empty and unique}{ \expand{\(x\) is empty}{ (\not\exists t)(t\in x) } \land \expand{\(x\) is unique}{ (\forall y)\expand{if \(y\) is empty then \(x=y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } } } }\]

This set is denoted \(\varnothing\).

The Axiom of Existence, Weaker Version There exists an empty set.

\[\expand{There exists an empty set.}{ (\exists x)\expand{\(x\) is empty}{ (\not\exists t)(t\in x) } }\]

Such a set is denoted \(\varnothing\).

All empty sets are equal.

\[\expand{All empty sets are equal.}{ (\forall x)\expand{if \(x\) is empty then it is unique}{ \expand{\(x\) is empty}{ (\not\exists t)(t\in x) } \implies \expand{\(x\) is unique}{ (\forall y)\expand{if \(y\) is empty then \(x=y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } } } }\]

Let x be an empty set.		premise
Let y be an empty set.		assumption
x and y are empty sets		conjunction introduction
\not\exists t : t\in x \land \not\exists t : t\in y		dfn of empty
\forall t : t\not\in x \land t\not\in y					dfn of universal quantifier
assume t\in x				assumption
t\not\in x \land t\in x		conjunction introduction
contradiction				law of non-contradiction
t\in y						law of explosion
now, assume t\in y			assumption
... t\in x					WLOG
x=y							dfn of equality

(recap: if \forall t : t\not\in x \land t\not\in y, then assuming t\in x, we can conclude t\in y, and assuming t\in y, we can conclude t\in x. thus x=y. In conclusion, if x is empty and there is another set y that is also empty, then x=y. The empty set must be unique.)

Every set includes an empty set.

\[\expand{Every set includes an empty set.}{ (\forall x)\expand{\(x\) includes an empty set}{ (\exists y)\expand{\(y\) is empty and \(x\) includes \(y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \land \expand{\(y\subseteq x\)}{ (\forall t)(t\in y\implies t\in x) } } } }\]

Tuples. Informal and not part of the system.

\begin{align} \varnothing &:= \{\}\\ \langle\rangle &:= \{\varnothing,\varnothing\} &&= \{\varnothing\}\\ \langle a\rangle &:= \{\langle\rangle,a\} &&= \bigl\{\{\varnothing\}, a\bigr\}\\ \langle a,b\rangle &:= \{\langle a\rangle,b\} &&= \Bigl\{\bigl\{\{\varnothing\}, a\bigr\}, b\Bigr\}\\ \langle a,b,c\rangle &:= \{\langle a,b\rangle,c\} &&= \biggl\{\Bigl\{\bigl\{\{\varnothing\}, a\bigr\}, b\Bigr\}, c\biggr\}\\ \langle a,b,c,d\rangle &:= \{\langle a,b,c\rangle,d\} &&= \Biggl\{\biggl\{\Bigl\{\bigl\{\{\varnothing\}, a\bigr\}, b\Bigr\}, c\biggr\}, d\Biggr\} \end{align}

Ordered Pairs. Formal and necessary.

\begin{align} (a,b) &:= \{\{a\},b\}\\ (a,b,c) &:= \bigl((a,b),c\bigr) = \Bigl\{\bigl\{(a,b)\bigr\},c\Bigr\} = \Bigl\{\bigl\{\bigl\{\{a\},b\bigr\}\bigr\},c\Bigr\}\\ (a,b,c,d) &:= \bigl((a,b,c),d\bigr) \end{align}
