Predicate Logic

A set is that which may own sets.

Given sets \(s\) and \(t\), the assertion that \(s\) owns \(t\) is a proposition, denoted \(t\in s\). We may also say that \(t\) is an element of, or a member of, \(s\).

We use the predicate \(t\not\in s\) to refer to the negation \(\lnot(t\in s)\).

To avoid ambiguity, we will not use the word contain to represent ownership. For historical reasons, this expression can be interpreted as an indication of ownership or one of inclusion.

The Principle of Bivalence, Predicate Version Given sets \(t\) and \(s\):

  • The Law of Excluded Middle It is always the case that \(t\) is an element of \(s\) or not an element of \(s\).

    \[(t\in s) \lor \lnot(t\in s)\]

  • The Law of Non-Contradiction It is never the case that \(t\) is both an element of \(s\) and not an element of \(s\).

    \[\lnot\bigl((t\in s) \land \lnot(t\in s)\bigr)\]

Given a set \(x\), the predicate \(\l P_x\) is a proposition that makes an assertion involving \(x\).

In the natural numbers, let \(\l P_n\) be the predicate \(n\) is even. Then \(\l P_3\) would be false, while \(\l P_6\) would be true.

Given a predicate \(\l P_x\), the assertion that there exists a set \(x\) that makes \(\l P_x\) true is itself a predicate, called an existential quantifier.

\[(\exists x)(\l P_x)\]

We use the predicate \((\not\exists x)(\l P_x)\) to refer to the negation \(\lnot\bigl((\exists x)(\l P_x)\bigr)\).

Given a predicate \(\l P_x\) and a set \(y\), the assertion that there exists an element \(x\) of \(y\) that makes \(\l P_x\) true is a predicate called a bound existential quantifier.

\[ (\exists x\in y)(\l P_x) \equiv (\exists x)(x\in y \land \l P_x) \]

Let \(\l P_x\) be the predicate \(x\in x\). Then if there exists some set \(x\) that is a member of itself, \((\exists x)(x\in x)\) would be true. (However, we will see later that there does not exist such a set.)

Given a predicate \(\l P_x\), the assertion that there does not exist any set \(x\) that makes \(\l P_x\) false is itself a predicate, \((\not\exists x)(\lnot\l P_x)\), called a universal quantifier. This is the assertion that \(\l P_x\) is true for any set \(x\).

\[(\forall x)(\l P_x)\]

Given a predicate \(\l P_x\) and a set \(y\), the assertion that \(\l P_x\) is true for all elements \(x\) of \(y\) is a predicate called a bound universal quantifier.

\[ (\forall x\in y)(\l P_x) \equiv (\forall x)(x\in y \implies \l P_x) \]

\[ \bigl((\forall x\in y)(\l P_x)\bigr) \iff \bigl((\not\exists x\in y)(\lnot\l P_x)\bigr) \]

\[ (\forall x\in y)(\l P_x)\\ (\forall x)(x\in y \implies \l P_x)\\ (\forall x)(x\not\in y \lor \l P_x)\\ (\forall x)(\lnot(x\in y \land \lnot\l P_x))\\ (\not\exists x)(x\in y \land \lnot\l P_x)\\ (\not\exists x\in y)(\lnot\l P_x)\\ \]

Combined Quantifiers

Given an existential quantifier \((\exists y)(\l P_{x,y})\), the existential quantifier \(\bigl(\exists x\bigr)\bigl((\exists y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\exists x)(\exists y)(\l P_{x,y})\]

Given a universal quantifier \((\forall y)(\l P_{x,y})\), the universal quantifier \(\bigl(\forall x\bigr)\bigl((\forall y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\forall x)(\forall y)(\l P_{x,y})\]

existential quantification is commutative

\[ \bigl((\exists x)(\exists y)(\l P_{x,y})\bigr) \equiv \bigl((\exists y)(\exists x)(\l P_{x,y})\bigr) \]

universal quantification is commutative

\[ \bigl((\forall x)(\forall y)(\l P_{x,y})\bigr) \equiv \bigl((\forall y)(\forall x)(\l P_{x,y})\bigr) \]

an existential disjunction is logically equivalent to a disjunction of existential quantifiers

\[ \bigl((\exists x)(\l P_x \lor \l Q_x)\bigr) \equiv \bigl((\exists a)(\l P_a) \lor (\exists b)(\l Q_b)\bigr) \]

a universal disjunction is logically equivalent to a disjunction of universal quantifiers

\[ \bigl((\forall x)(\l P_x \lor \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \lor (\forall b)(\l Q_b)\bigr) \]

an existential conjunction logically implies a conjunction of existential quantifiers

\[ \bigl((\exists x)(\l P_x \land \l Q_x)\bigr) \therefore \bigl((\exists a)(\l P_a) \land (\exists b)(\l Q_b)\bigr) \]

a universal conjunction is logically equivalent to a conjunction of universal quantifiers

\[ \bigl((\forall x)(\l P_x \land \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \land (\forall b)(\l Q_b)\bigr) \]

an existential implication logically implies an implication of existential quantifiers

\[ \bigl((\exists x)(\l P_x \implies \l Q_x)\bigr) \therefore \bigl((\exists a)(\l P_a) \implies (\exists b)(\l Q_b)\bigr) \]

a universal implication is logically equivalent to an implication of universal quantifiers

\[ \bigl((\forall x)(\l P_x \implies \l Q_x)\bigr) \equiv \bigl((\forall a)(\l P_a) \implies (\forall b)(\l Q_b)\bigr) \]

A universal implication is logically equivalent to the contrapositive of existential quantifiers

\[ \bigl((\forall x)(\l P_x \implies \l Q_x)\bigr) \equiv \bigl((\exists a)(\lnot\l Q_a) \implies (\exists b)(\lnot\l P_b)\bigr) \]

an implication of existential quantifiers is logically equivalent to a universal implication whose consequent is an existential quantifier

\[ \bigl((\exists a)(\l P_a) \implies (\exists b)(\l Q_b)\bigr) \equiv \Bigl(\bigl(\forall x\bigr)\bigl(\l P_x \implies (\exists y)(\l Q_{x,y})\bigr)\Bigr) \]

\[ \Bigl(\bigl(\exists x\bigr)\bigl((\forall y)(\l P_{x,y})\bigr)\Bigr) \therefore \Bigl(\bigl(\forall y\bigr)\bigl((\exists x)(\l P_{x,y})\bigr)\Bigr) \]

Equality and Inclusion

The Intensionality Definition A set is equal to another set iff those sets own each other's elements.

We use the predicate \(x=y\) to describe the assertion that a set \(x\) is equal to a set \(y\). We define this predicate as:

\[ \bigl(x=y\bigr) \equiv \bigl((\forall t)(t\in x \iff t\in y)\bigr) \]

We use the predicate \(x\not= y\) to refer to the negation \(\lnot(x=y)\).

The Axiom of Extensionality Equal elements are members of the same sets.

\[ (\forall x)(\forall y) \expand{if \(x=y\) then \(x\) and \(y\) are in the same sets}{ \expand{\(x=y\)}{ (\forall t)(t\in x\iff t\in y) } \implies \left((\forall z)(x\in z \iff y\in z)\right) } \]

Intensionality states that sets that are equal have exactly the same members. In other words, equal sets are identical. One very important interpretation of this is that a set is determined uniquely its elements, so if sets with exactly the same elements have different definitions, names, or rearrangements, those sets are all one in the same. This may seem obvious, but a more fascinating fact about equal sets follows below.

By Extensionality, all sets are definite and distinguishable. This means that given a set, one should be able to decide whether any particular element would belong to that set; and that any two elements of the same set are unequal.

A set \(x\) is a subset of a set \(y\) iff \(y\) owns all the elements of \(x\).

We use the predicate \(x\subseteq y\) to describe the assertion that \(x\) is a subset of \(y\). We define this predicate as:

\[ \bigl(x\subseteq y\bigr) \equiv \bigl((\forall t)(t\in x \implies t\in y)\bigr) \]

We also say \(x\) is included in \(y\), or \(y\) includes \(x\).

A set \(x\) is a proper subset of a set \(y\) iff \(x\subseteq y\) and \(x\not= y\).

We use the predicate \(x\subset y\) to describe the assertion that \(x\) is a proper subset of \(y\). We define this predicate as:

\[ \bigl(x\subset y\bigr) \equiv \bigl(\expand{\(x\subseteq y\)}{ (\forall t)(t\in x \implies t\in y) }\land\expand{\(x\not=y\)}{ (\exists t)(t\in x \xor t\in y) }\bigr) \]

We also say \(x\) is strictly included in \(y\), or \(y\) strictly includes \(x\).

Remember the distinction between \(y\) owns \(x\) and \(y\) includes \(x\).

In the former expression, we are trying to determine if \(x\) is an element of \(y\). To do so, we look at the set \(x\) as a single object, and then check to see if this object is among the members of the set \(y\). If so, \(x\in y\).

In the latter expression, we are trying to determine if \(x\) is a subset of \(y\). To do this, we must open up the set \(x\), examine its various elements, and then check to see if the set \(y\) owns every one of those elements. If so, \(x\subseteq y\).

In each case, we will avoid the term contain because of its ambiguous meaning in different sources. (A set could contain its elements or it could contain its subsets.)

\[ (\forall x)\expand{\(x=x\)}{ (\forall t)(t\in x \iff t\in x) } \]

Uniqueness Quantifiers

A uniqueness quantifier is an assertion that there exists a particular set, and that is the only one that exists. Essentially, by stating a uniqueness quantifier, we assert that there exists a set that exhibits a certain characteristic, and all sets that exhibit that characteristic are equal.

define: existence in the first statement; uniqueness in the second

There exists an x such that P_x is true, and for all x such that P_x is true, P_y is true if and only if x=y.

\[ \biggl((\exists!x)(\l P_x)\biggr) \equiv \biggl( \Bigl((\exists x)(\l P_x)\Bigr) \land \Bigl( \bigl(\forall x\bigr) \bigl(\l P_x \implies (\forall y)(\l P_y \iff x=y)\bigr) \Bigr) \Biggr) \]

Prove: There exists an x such that P_x is true, and for any two x and y, if P_x and P_y are true, then x=y.

\[ \biggl((\exists!x)(\l P_x)\biggr) \equiv \biggl( \Bigl((\exists x)(\l P_x)\Bigr) \land \Bigl( \bigl(\forall x\bigr)\bigl(\forall y\bigr) \bigl( (\l P_x \land \l P_y) \implies (x=y) \bigr) \Bigr) \biggr) \]

prove: existence and uniqueness in one statement

\[ \bigl((\exists!x)(\l P_x)\bigr) \equiv \Bigl( \bigl(\exists x\bigr) \bigl(\l P_x \land (\forall y)(\l P_y \iff x=y)\bigr) \Bigr) \]

The Axioms of Set Theory

The Seven Pillars of Creation

The seven axioms below serve as the foundation of all mathematics. The first six tell us what sets may be brought into existence, and the seventh specifies a important characteristic of sets that do exist.

Existence

A set is called empty, denoted \(\varnothing\), iff it has no members.

\[ \left(t\in\varnothing\right) \equiv \bot \]

We use the predicate \(x\doteq\varnothing\) to refer to the assertion that \(x\) is empty, and \(x\not\doteq\varnothing\) for the negation.

The Axiom of Existence There exists an empty set.

\[ \expand{\(\exists\varnothing\)}{ (\exists x)\expand{\(x\doteq\varnothing\)}{ (\forall t)\expand{\(t\not\in x\)}{\lnot(t\in x)} } } \]

The empty set is unique.

\[ (\forall x)(\forall y)\left( \left( \expand{\(x\doteq\varnothing\)}{ (\forall t)\expand{\(t\not\in x\)}{\lnot(t\in x)} } \land \expand{\(y\doteq\varnothing\)}{ (\forall t)\expand{\(t\not\in y\)}{\lnot(t\in y)} } \right) \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } \right) \]

To describe any set \(x\) as empty, we assert that \((\forall t)(t\not\in x)\) and use the notation \(x\doteq\varnothing\) as shorthand. But we now see that the notation has a quite literal meaning. That is, any set \(x\) that is empty is literally equal to the empty set \(\varnothing\). We can verify this with The Intensionality Definition. This goes to show that all empty sets are equal, as proven above.

In our informal discussions, we can reconcile this idea by replacing the shorthand notation with equality.

\[x\doteq\varnothing \equiv x=\varnothing\]

However in rigorous well-formed formulae we cannot use \(x=\varnothing\) because \(\varnothing\) is not technically a symbol in predicate logic.

Pair

Given sets \(x\) and \(y\), a set is called a pair of \(x\) and \(y\), denoted \(\left\{x,y\right\}\), iff it owns exactly \(x\) and \(y\).

\[ \left(w\in \{x,y\}\right) \equiv \left( (w=x) \lor (w=y) \right) \]

We use the predicate \(z\doteq\{x,y\}\) to refer to the assertion that \(z\) is a pair of \(x\) and \(y\).

The Axiom of Pair For every two sets, there exists a set that owns exactly them both.

\[ (\forall x)(\forall y)\expand{\(\exists\left\{x,y\right\}\)}{ (\exists z)\expand{\(z\doteq\left\{x,y\right\}\)}{ (\forall w)\left( (w\in z) \iff \left( \expand{\(w=x\)}{ (\forall t)(t\in w \iff t\in x) } \lor \expand{\(w=y\)}{ (\forall t)(t\in w \iff t\in y) } \right) \right) } } \]

A pair constructed from two given sets is unique.

\[ (\forall x)(\forall y)\expand{\(\left\{x,y\right\}\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteq\left\{x,y\right\}\)}{ (\forall z)\left( (z\in a) \iff \left( \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } \right) \right) } \land \expand{\(b\doteq\left\{x,y\right\}\)}{ (\forall z)\left( (z\in b) \iff \left( \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } \right) \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

Power

Given a set \(x\), a set is called a power set of \(x\), denoted \(\wp x\), iff it owns exactly all subsets of \(x\).

\[ \left(z\in \wp x\right) \equiv \left(z\subseteq x\right) \]

The Axiom of Power There exists a set that owns exactly all subsets of any given set.

\[ (\forall x)\expand{\(\exists \wp x\)}{ (\exists y)\expand{\(y\doteq\wp x\)}{ (\forall z)\left( z\in y \iff \expand{\(z\subseteq x\)}{ (\forall t)(t\in z \implies t\in x) } \right) } } \]

The power set constructed from a given set is unique.

\[ (\forall x)\expand{\(\wp x\) is unique}{ (\forall a)(\forall b)\left( \left( \expand{\(a\doteq\wp x\)}{ (\forall y)\left( y\in a \iff \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } \right) } \land \expand{\(b\doteq\wp x\)}{ (\forall y)\left( y\in b \iff \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } \right) } \right) \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } \right) } \]

Union

Given a set \(x\), a set is called a union of \(x\), denoted \(\bigcup x\), iff it owns exactly any members belonging to any elements of \(x\).

\[ \left(z\in \bigcup x\right) \equiv \left( (\exists w)(w\in x \land z\in w) \right) \]

Given the pair \(\left\{a,b\right\}\), the union \(\bigcup\left\{a,b\right\}\) is called, for simplicity, the union of \(a\) and \(b\). We use the shorthand notation \(a\cup b\) instead.

The Axiom of Union There exists a set that owns exactly any element belonging to any element of a given nonempty set.

\[ (\forall x)\left( \expand{\(x\not=\varnothing\)}{ (\exists t)(t\in x) } \implies \expand{\(\exists\bigcup x\)}{ (\exists y)\expand{\(y\doteq\bigcup x\)}{ (\forall z)\left( z\in y \iff \expand{\(z\) belongs to an element of \(x\)}{ (\exists w)(w\in x \land z\in w) } \right) } } \right) \]

The union constructed from a set is unique.

Intersection

Given a set \(x\), a set is called an intersection of \(x\), denoted \(\bigcap x\), iff it owns exactly all members belonging to all elements of \(x\).

\[ \left(z\in \bigcap x\right) \equiv \left( (\forall w)(w\in x \implies z\in w) \right) \]

Given the pair \(\left\{a,b\right\}\), the intersection \(\bigcap\left\{a,b\right\}\) is called, for simplicity, the intersection of \(a\) and \(b\). We use the shorthand notation \(a\cap b\) instead.

The Axiom of Intersection There exists a set that owns exactly all elements belonging to all elements of a given nonempty set.

\[ (\forall x)\left( \expand{\(x\not=\varnothing\)}{ (\exists t)(t\in x) } \implies \expand{\(\exists\bigcap x\)}{ (\exists y)\expand{\(y\doteq\bigcap x\)}{ (\forall z)\left( z\in y \iff \expand{\(z\) belongs to all elements of \(x\)}{ (\forall w)(w\in x \implies z\in w) } \right) } } \right) \]

The intersection constructed from a set is unique.

Infinity

A set is called inductive iff it is nonempty and each of its elements is owned by another element of the set.

The Axiom of Infinity There exists a nonempty set of which each element is owned by another element of this set.

\[ (\exists x)\expand{\(x\) is inductive}{ \expand{\(x\not=\varnothing\)}{ (\exists t)(t\in x) } \land \expand{each element of \(x\) is owned by of another element of \(x\)}{ (\forall y)\left( (y\in x) \implies \expand{\(y\) is a member of another element of \(x\)}{ (\exists z)\left( (z\in x) \land (y\in z) \right) } \right) } } \]

At this point, we do not have a formal definition for a set that is infinite, but informally we can think of infinite sets as having never-ending elements: there are more elements than we can count. One way to achieve this is to assert the existence of an inductive set. Indeed, it is true that all inductive sets are infinite. The converse is left as a cliffhanger.

Regularity

The intent of The Axiom of Infinity is that there exists an infinite set; that is, a set with an unlimited number of elements. However, dfn-inductive only requires that an inductive set be nonempty, and own at least one element for every element it owns.

For example, it is not unreasonable to consider the sets \(a\) and \(b\) such that \(a\in b\) and \(b\in a\). Formally, we are not guaranteed the existence of such sets, but let's hypothesize that they do exist. Certainly, the pair \(\{a,b\}\) exists, and indeed it does meet the requirements of dfn-inductive: \(\{a,b\}\) is nonempty, and for every \(x\in \{a,b\}\), there exists a \(y\in \{a,b\}\) such that \(x\in y\).

It is to our disappointment that a set such as \(\{a,b\}\) above is inductive but is not infinite (satisfying our intuitive understanding of infinite sets). Thus we must create a new rule that does not allow sets to own themselves, or to own elements that own themselves, or so on. Essentially, we cannot allow the chain of ownership to be cyclical. This will ensure that for any element in an inductive set, we must have a new element, one that hasn't been mentioned before, in the set. This will satisfy our intuitive notion of a set with more elements than we can count.

A set is called regular iff it is empty or it has an element that does not own another element of the set.

The Axiom of Regularity Every nonempty set has an element that does not own another element of the set.

\[ (\forall x)\expand{\(x\) is regular}{ \expand{\(x\not=\varnothing\)}{ (\exists t)(t\in x) } \implies \expand{there exists an element of \(x\) that does not own another element of \(x\)}{ (\exists y)\left( (y\in x) \land \expand{no members of \(y\) are elements of \(x\)}{ (\forall z)\left( (z\in x) \implies \expand{\(z\not\in y\)}{\lnot(z\in y)} \right) } \right) } } \]

A singleton constructed from one set is unique.

The element of a singleton is unique.

Specification Given a set, there exists a subset of elements that adhere to any given predicate.

For any set and a given predicate, there exists a set owning elements of that set, under that predicate.

(\forall B)(\exists A)(\forall x)\bigl(x\in A \iff x\in B \land \l P_x\bigr)
We are given that the collection~$\mathcal{S}$ is nonempty. Let the set~$A$ be some fixed member of $\mathcal{S}$. \Aref{A:specification} guarantees there exists some set $T$ such that $$\forall x : \Bigl(x \in T \iff x \in A \land \bigl(\forall B \in \mathcal{S} : B\not=A : (x \in B)\bigr)\Bigr)$$ In other words, $x$ is an element of $T$ iff $x$ is an element of $A$ and every other set in $\mathcal{S}$. It follows that $x$ belongs to every element of $\mathcal{S}$. Then we define $T$ to be the intersection of the collection~$\mathcal{S}$.% \Aref{A:extensionality} guarantees $T$ is unique.
