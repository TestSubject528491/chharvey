Section II.A: abc

A set is that which may contain elements. These elements, in turn, are also sets.

There are a number of other ways to express set containment.
\[t\in S\] \[S\owns t\]
\(t\) is an element of \(S\) \(S\) contains \(t\)
\(t\) is a member of \(S\) \(S\) is a possessor of \(t\)

If \(S\) does not contain \(t\), we state, \(\lnot(x\in X)\), \(x\not\in X\), or \(X\not\owns x\).

A set \(A\) is equal to to a set \(B\) if and only if \(A\) and \(B\) contain each other's elements. That is,

\[(A=B) \text{ if and only if } (\forall t)(t\in A \iff t\in B)\]

This definition is often called The Axiom of Extensionality.

The Principle of Bivalence, Set Theory Version

Given sets \(t\) and \(S\):

  • The Law of Excluded Middle It is always the case that \(t\) is an element of or not an element of \(S\).

    \[(t\in S) \lor (t\not\in S)\]

  • The Law of Non-Contradiction It is never the case that \(t\) is both an element of and not an element of \(S\).

    \[\lnot((t\in S) \land (t\not\in S))\]

The Axiom of Existence There exists an empty set, and that set is unique.

\[\expand{There exists an empty set, and that set is unique.}{ (\exists x)\expand{\(x\) is empty and unique}{ \expand{\(x\) is empty}{ (\not\exists t)(t\in x) } \land \expand{\(x\) is unique}{ (\forall y)\expand{if \(y\) is empty then \(x=y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } } } }\]

This set is denoted \(\varnothing\).

Every set includes an empty set.

\[\expand{Every set includes an empty set.}{ (\forall x)\expand{\(x\) includes an empty set}{ (\exists y)\expand{\(y\) is empty and \(x\) includes \(y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \land \expand{\(y\subseteq x\)}{ (\forall t)(t\in y\implies t\in x) } } } }\]

Tuples. Informal and not part of the system.

\begin{align} \langle\rangle &:= \{\varnothing,\varnothing\} = \{\varnothing\} &&= 1\\ \langle a\rangle &:= \{\langle\rangle,a\} &&= \{1, a\}\\ \langle a,b\rangle &:= \{\langle a\rangle,b\} &&= \bigl\{\{1, a\}, b\bigr\}\\ \langle a,b,c\rangle &:= \{\langle a,b\rangle,c\} &&= \Bigl\{\bigl\{\{1, a\}, b\bigr\},c\Bigr\}\\ \langle a,b,c,d\rangle &:= \{\langle a,b,c\rangle,d\} \end{align}

Ordered Pairs. Formal and necessary.

\begin{align} (a,b) &:= \{\{a\},b\}\\ (a,b,c) &:= \bigl((a,b),c\bigr) = \{\{(a,b)\},c\} = \Bigl\{\bigl\{\bigl\{\{a\},b\bigr\}\bigr\},c\Bigr\}\\ (a,b,c,d) &:= \bigl((a,b,c),d\bigr) \end{align}
