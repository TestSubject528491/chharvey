Predicate Logic

A set is that which may own sets.

Given sets \(s\) and \(t\), the assertion that \(s\) owns \(t\) is a proposition, denoted \(t\in s\). We may also say that \(t\) is an element of, or a member of, \(s\).

We use the predicate \(t\not\in s\) to refer to the negation \(\lnot(t\in s)\).

To avoid ambiguity, we will not use the word contain to represent ownership. For historical reasons, this expression can be interpreted as an indication of ownership or one of inclusion.

The Principle of Bivalence, Predicate Version Given sets \(t\) and \(s\):

  • The Law of Excluded Middle It is always the case that \(t\) is an element of \(s\) or not an element of \(s\).

    \[(t\in s) \lor (t\not\in s)\]

  • The Law of Non-Contradiction It is never the case that \(t\) is both an element of \(s\) and not an element of \(s\).

    \[\lnot\bigl((t\in s) \land (t\not\in s)\bigr)\]

use Principle of Bivalence, Proposition Version

Predicates and Quantifiers

Given a set \(x\), the predicate \(\l P_x\) is a proposition that makes an assertion involving \(x\).

In the natural numbers, let \(\l P_n\) be the predicate \(n\) is even. Then \(\l P_3\) would be false, while \(\l P_6\) would be true.

Given a predicate \(\l P_x\), the assertion that there exists a set \(x\) that makes \(\l P_x\) true is itself a predicate, called an existential quantifier.

\[(\exists x)(\l P_x)\]

We use the predicate \((\not\exists x)(\l P_x)\) to refer to the negation \(\lnot\bigl((\exists x)(\l P_x)\bigr)\).

Given a predicate \(\l P_x\) and a set \(y\), the assertion that there exists an element \(x\) of \(y\) that makes \(\l P_x\) true is a predicate called a bound existential quantifier.

\[ (\exists x\in y)(\l P_x) \equiv (\exists x)(x\in y \land \l P_x) \]

Let \(\l P_x\) be the predicate \(x\in x\). Then if there exists some set \(x\) that is a member of itself, \((\exists x)(x\in x)\) would be true. (However, we will see later that there does not exist such a set.)

Given a predicate \(\l P_x\), the assertion that there does not exist any set \(x\) that makes \(\l P_x\) false is itself a predicate, \((\not\exists x)(\lnot\l P_x)\), called a universal quantifier. This is the assertion that \(\l P_x\) is true for any set \(x\).

\[(\forall x)(\l P_x)\]

Given a predicate \(\l P_x\) and a set \(y\), the assertion that \(\l P_x\) is true for all elements \(x\) of \(y\) is a predicate called a bound universal quantifier.

\[ (\forall x\in y)(\l P_x) \equiv (\forall x)(x\in y \implies \l P_x) \]

Combined Quantifiers

Given an existential quantifier \((\exists y)(\l P_{x,y})\), the existential quantifier \(\bigl(\exists x\bigr)\bigl((\exists y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\exists x)(\exists y)(\l P_{x,y})\]

Given a universal quantifier \((\forall y)(\l P_{x,y})\), the universal quantifier \(\bigl(\forall x\bigr)\bigl((\forall y)(\l P_{x,y})\bigr)\) is described by the predicate:

\[(\forall x)(\forall y)(\l P_{x,y})\]

Uniqueness Quantifiers

A uniqueness statement is a statement that asserts that that particular set is the only one that exists. For example, there is one and only one solution to a given problem.

existence and uniqueness in one statement
\[ (\exists!x)(\l P_x) \text{ is defined by } (\exists x)(\l P_x \land (\forall y)(\l P_y \iff x=y)) \]
existence in the first statement; uniqueness in the second
\[ (\exists!x)(\l P_x) \text{ is defined by } (\exists x)(\l P_x) \land (\forall x) (\l P_x \implies (\forall y)(\l P_y \iff x=y))\\ \]
\[ (\exists!x)(\l P_x) \text{ proven by } (\exists x)(\l P_x) \land (\forall x)(\forall y) ((\l P_x \land \l P_y) \implies (x=y)) \]

Equality and Inclusion

A set is equal to another set if and only if those sets own each other's elements.

We use the predicate \(x=y\) to describe the assertion that a set \(x\) is equal to a set \(y\). We define this predicate as:

\[ (x=y) \equiv (\forall t)(t\in x \iff t\in y) \]

This definition is often called The Axiom of Extensionality.

We use the predicate \(x\not= y\) to refer to the negation \(\lnot(x=y)\).

One very important interpretation of The Axiom of Extensionality is that sets are determined completely and only by their elements. If sets with exactly the same elements have different definitions, names, or rearrangements (orders), those sets are all one in the same.

Axioms of Equality

Equal elements must be members of the same sets.

\[ (\forall x)(\forall y) \expand{if \(x=y\) then \(x\) and \(y\) are in the same sets}{ (x=y) \implies \left((\forall z)(x\in z \iff y\in z)\right) } \]

All sets are definite and distinguishable. This means that given a set, one should be able to decide whether any particular element would belong to that set; and that any two elements of the same set are different. The term different is not yet explicitly defined.

Equality is an equivalence relation on any set.

\(\left(x,y\right)\in\mathscr{E}\), where \(\mathscr{E}\) represents the subset of \(A\times A\) specified by \(\left(x,y\right)\in\mathscr{E} \iff \left(x,y\right)\in A^2 \land \left(\forall t\right)\left(t\in x\iff t\in y\right)\) for any set \(A\).

Equality is reflexive.

\[\expand{Any set is equal to itself.}{ (\forall x)\expand{\(x=x\)}{ (\forall t)(t\in x \iff t\in x) } }\]

\[ (\forall x)\expand{\(x=x\)}{ (\forall t)(t\in x \iff t\in x) } \]

use \(\l p \iff \l p \therefore \top\)

Equality is symmetric.

\[\expand{One set is equal to another set if and only if that other set is equal to the first set.}{ (\forall x)(\forall y)\left( \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } \implies \expand{\(y=x\)}{ (\forall t)(t\in y \iff t\in x) } \right) }\]

Ergo, when \(x=y\), instead of saying, \(x\) is equal to \(y\), we may just as well say, \(x\) and \(y\) are equal.

use \(\l p \iff \l q \therefore \l q \iff \l p\)

Equality is transitive.

\[\expand{If one set is equal to a second set, and that second set is equal to a third set, then all three sets are equal.}{ (\forall x)(\forall y)(\forall z)\left( \left( \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } \land \expand{\(y=z\)}{ (\forall t)(t\in y \iff t\in z) } \right) \implies \expand{\(x=z\)}{ (\forall t)(t\in x \iff t\in z) } \right) }\]

use \(\l p \iff \l q \land \l q \iff \l r \therefore \l p \iff \l r\)

A set \(x\) is a subset of a set \(y\) if and only if \(y\) owns all the elements of \(x\).

We use the predicate \(x\subseteq y\) to describe the assertion that \(x\) is a subset of \(y\). We define this assertion as:

\[(\forall t)(t\in x \implies t\in y)\]

We also say \(x\) is included in \(y\), or \(y\) includes \(x\).

Inclusion is reflexive.

Inclusion is antisymmetric?.

\[\expand{Two sets are equal if and only if those sets include each other.}{ (\forall x)(\forall y)\expand{\(x=y\) iff they include each other}{ \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } \iff \expand{\(x\) and \(y\) include each other}{ \expand{\(x\subseteq y\)}{ (\forall t)(t\in x \implies t\in y) } \land \expand{\(y\subseteq x\)}{ (\forall t)(t\in y \implies t\in x) } } } }\]

Inclusion is transitive.

The Axioms of Set Theory

The Axioms of Creation

The six axioms below tell us what sets may be brought into existence. There are other axioms, but they merely make statements about sets that already exist.

The Axiom of Existence There exists an empty set.

\[\expand{There exists an empty set.}{ (\exists x)\expand{\(x\) is empty}{ (\forall y)(y\not\in x) } }\]

The Axiom of Pair For every two sets, there exists a set that owns exactly them both.

\[\expand{For every two sets, there exists a set that owns exactly them both.}{ (\forall x)(\forall y)\expand{there exists a set that owns exactly \(x\) and \(y\)}{ (\exists z)\expand{\(z\) owns \(x\) and \(y\) and no other sets}{ (\forall w)\expand {\(z\) owns \(w\) if and only if \(w\) is \(x\) or \(y\)}{ (w\in z) \iff \expand{\(w=x\)}{ (\forall t)(t\in w \iff t\in x) } \lor \expand{\(w=y\)}{ (\forall t)(t\in w \iff t\in y) } } } } }\]

The Axiom of Specification Given a set, there exists a subset of elements that adhere to any given predicate.

For any set and a given predicate, there exists a set owning elements of that set, under that predicate.

(\forall B)(\exists A)(\forall x)\bigl(x\in A \iff x\in B \land \l P_x\bigr)

The Axiom of Union For every set, there exists a set which owns exactly any element belonging to any element of that set.

The Axiom of Power There exists a set that has, as elements, exactly all subsets of any given set.

The Axiom of Infinity There exists a nonempty set such that each element of this set is a member of another element of this set.

\[\expand{There exists a nonempty set such that each element of this set is a member of another element of this set.}{ (\exists x)\expand{\(x\) is nonempty and each element of \(x\) is a member of another element of \(x\)}{ \expand{\(x\) is nonempty}{ (\exists t)(t\in x) } \land \expand{each element of \(x\) is a member of another element of \(x\)}{ (\forall y)\expand{if \(y\) is an element of \(x\) then \(y\) is a member of another element of \(x\)}{ (y\in x) \implies \expand{there is another element of \(x\) that owns \(y\)}{ (\exists z)\expand{\(z\) is an element of \(x\) that owns \(y\)}{ (z\in x) \land (y\in z) } } } } } }\]

	the intent of this axiom is that there exists an 'infinite' set; that is, a set with an 'unlimited number' of elements.
	without the Axiom of Regularity, the Axiom of Infinity allows for such a set that meets the conditions stated above and that has only 2 elements: for example, the set c = {a,b}, with a ∈ b and b ∈ a. (indeed, c is nonempty, and each element of c is owned by at least one other element of c.)
	Thus, we need the Axiom of Regularity to prevent such a set from existing:
	With the Axiom of Regularity, c cannot be a set because for all elements x ∈ c, we have c∩x is nonempty.

Other Axioms

The Axiom of Foundation Membership is antireflexive.

\[\expand{No set is a member of itself.}{ (\forall x)(x\not\in x) }\]

The empty set is unique.

\[\expand{The empty set is unique.}{ (\forall x)\expand{if \(x\) is empty then it is unique}{ \expand{\(x\) is empty}{ (\forall t)(t\not\in x) } \implies \expand{\(x\) is unique}{ (\forall y)\expand{\(y\) is empty iff \(x=y\)}{ \expand{\(y\) is empty}{ (\forall t)(t\not\in y) } \iff \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } } } }\]

\[\expand{All empty sets are equal.}{ (\forall x)(\forall y)\expand{if \(x\) and \(y\) are empty then they are equal}{ \expand{\(x\) and \(y\) are empty}{ (\forall t)(t\not\in x \land t\not\in y) } \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } }\]

The empty set is denoted \(\varnothing\).

Let x be an empty set.		premise
Let y be an empty set.		assumption
x and y are empty sets		conjunction introduction
\not\exists t : t\in x \land \not\exists t : t\in y		dfn of empty
\forall t : t\not\in x \land t\not\in y					dfn of universal quantifier
assume t\in x				assumption
t\not\in x \land t\in x		conjunction introduction
contradiction				law of non-contradiction
t\in y						law of explosion
now, assume t\in y			assumption
... t\in x					WLOG
x=y							dfn of equality

(recap: if \forall t : t\not\in x \land t\not\in y, then assuming t\in x, we can conclude t\in y, and assuming t\in y, we can conclude t\in x. thus x=y. In conclusion, if x is empty and there is another set y that is also empty, then x=y. The empty set must be unique.)

A pair constructed from two sets is unique.

\[\expand{A pair constructed from two sets is unique.}{ (\forall x)(\forall y)\expand{all pairs that own only \(x\) and \(y\) are equal}{ (\forall a)(\forall b)\expand{if \(a\) and \(b\) are pairs of \(x\) and \(y\) then \(a\) and \(b\) are equal}{ \expand{\(a\) and \(b\) own \(x\) and \(y\) and no other sets}{ \expand{\(a\) owns only \(x\) and \(y\)}{ (\forall z)\expand {\(a\) owns \(z\) if and only if \(z\) is \(x\) or \(y\)}{ (z\in a) \iff \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } } } \land \expand{\(b\) owns only \(x\) and \(y\)}{ (\forall z)\expand{\(b\) owns \(z\) if and only if \(z\) is \(x\) or \(y\)}{ (z\in b) \iff \expand{\(z=x\)}{ (\forall t)(t\in z \iff t\in x) } \lor \expand{\(z=y\)}{ (\forall t)(t\in z \iff t\in y) } } } } \implies \expand{\(a=b\)}{ (\forall t)(t\in a \iff t\in b) } } } }\]

Given sets \(x\) and \(y\), the set that contains only them both is denoted \(\left\{x,y\right\}\).

A singleton is a set that owns exactly one element.

A singleton constructed from one set is unique.

This theorem shows the well-definedness of the singleton function: if two singletons are unequal, their elements must be unequal.

The element of a singleton is unique.

This theorem shows the singleton function is injective: if two singletons are equal, their elements must be equal.

There is no set of which every set is a member.

\[\expand{There is no universal set.}{ (\forall x)\expand{there is a set that is not a member of \(x\)}{ (\exists y)(y\not\in x) } }\]

Two ordered pairs are equal if and only if their corresponding components are equal.

Every set includes an empty set.

\[\expand{Every set includes an empty set.}{ (\forall x)\expand{\(x\) includes an empty set}{ (\exists y)\expand{\(y\) is empty and \(x\) includes \(y\)}{ \expand{\(y\) is empty}{ (\forall t)(t\not\in y) } \land \expand{\(y\subseteq x\)}{ (\forall t)(t\in y\implies t\in x) } } } }\]

\[\expand{Every set includes an empty set.}{ (\forall x)\expand{\(x\) includes all empty sets}{ (\forall y)\expand{if \(y\) is empty then \(x\) includes \(y\)}{ \expand{\(y\) is empty}{ (\forall t)(t\not\in y) } \implies \expand{\(y\subseteq x\)}{ (\forall t)(t\in y\implies t\in x) } } } }\]

The antecedent $\forall x : x\in\emptyset$ is false, so the implication $\forall S : \forall x : (x \in \emptyset \implies x \in S)$ (from \Dref{D:subset}) is \emph{\indexthis{vacuous}ly} true. Thus $\emptyset \subseteq S$. It is because of this proof that the Empty Set is sometimes called the vacuous\indexterm{vacuous!set|seeonly{Empty Set}} set.
