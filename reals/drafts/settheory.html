A set is that which may contain elements. These elements, in turn, are also sets.

If a set \(S\) contains a set \(t\), we state, \(t\in S\) (\(t\) is an element of \(S\)), or alternatively, \(S\owns t\) (\(S\) contains \(t\)). If a set \(S\) does not contain a set \(t\), we state,\lnot\(x\in X\), \(x\not\in X\) or, \(X\not\owns x\).

Definition A set is a collection of objects. The objects in a set are called its elements. If \(X\) is a set and \(x\) is an object, then we state \(x\in X\) (or \(X\ni x\)) if and only if \(x\) is an element of \(X\). Otherwise, we state \(x\notin X\) (or \(X\not\ni x\)).

\(x\in X\)\(X\ni x\)
\(x\) is an element of \(X\) \(X\) owns \(x\)
\(x\) is a member of \(X\) \(X\) is a possessor of \(x\)

The Axiom of Existence There exists an empty set, and that set is unique.

\[\expand{There exists an empty set, and that set is unique.}{ (\exists x)\expand{\(x\) is empty and unique}{ \expand{\(x\) is empty}{ (\not\exists t)(t\in x) } \land \expand{\(x\) is unique}{ (\forall y)\expand{if \(y\) is empty then \(x=y\)}{ \expand{\(y\) is empty}{ (\exists t)(t\in y) } \implies \expand{\(x=y\)}{ (\forall t)(t\in x \iff t\in y) } } } } }\]

This set is denoted \(\varnothing\).

The Principle of Bivalence, Set Theory Version It is always the case that a set is an element of, or not an element of, another set, and it is never the case that a set is both an element of, and not an element of, another set.

\[ x\in X \lor x\not\in X \\ \lnot(x\in X \land x\not\in X) \]

\[ \forall X : (\forall x: \lnot(x\in X \iff x\not\in X)) \]

Axiom 2 (Axiom of Extensionality)

\[ \forall (X,Y)\in\mathcal{X}^2 : \bigl((X,Y)\in\mathscr{E}_\mathcal{X} \iff \forall t : (t\in X \iff t\in Y)\bigr) \]

Every set contains an empty set.

\[\expand{Every set contains an empty set.}{ (\forall x)\expand{\(x\) contains an empty set}{ (\exists y)\expand{\(y\) is empty and \(y\subset x\)}{} } }\]

\[\forall x ((\forall t)(t\not\in x) \implies \subseteq X)\]

Since the empty set has no elements, \(t\in\varnothing \implies t\in X\) by trivial implication. By definition of subset, \(\varnothing \subseteq X\).
