An Axiomatic Approach to the Continuum
Part II: Predicate Logic

Definition: predicate P_x

Given a set x, the predicate P_x is a proposition whose truth value is determined by x.

Let \(\mathsf{p}\) be a proposition, and let \(P\) and \(p\) be sets. Then

\[\mathsf{p}(x) \iff x\in X\]

example: in the natural numbers, let P_x be represented by the proposition, x is even. Then P_3 would be false, while P_6 would be true.

Definition: existential quantifier

Given a set S and elements a, b, c, d,..., and so on, of S, and given a predicate P_x, the existential quantifier (\exists x)(P_x) is defined by the predicate \mathsf{P}_a \lor \mathsf{P}_b \lor \mathsf{P}_c \lor ..., and so on

Definition Given a proposition \(\mathsf{p}\), the assertion that there exists an object \(x\) such that \(\mathsf{p}\) holds is itself proposition, denoted

\[\exists x : (\mathsf{p})\]

The symbol \(\exists\) is called the existential quantifier.

Definition: universal quantifier

Given a predicate \mathsf{P}_x, the universal quantifier (\forall x)(\mathsf{P}_x) is defined by the negation

\lnot((\exists x)(\lnot\mathsf{P}_x)) sometimes denoted (\notexists x)(\lnot\mathsf{P}_x)

Definition 

	definitions: bound quantifiers
	
		(\exists x\in S)(P_x) is defined by (\exists x)(x\in S \land P_x)
		
		(\forall x\in S)(P_x) is defined by (\forall x)(x\in S \implies P_x)
	
	
		Definition: uniqueness quantifier
		(\exists!x)(P_x) defined by
		(\exists x)(P_x \land (\forall y)(P_y \iff x=y))
	
		combined quantifiers
		
		prove:
		(\exists x)((\forall y)(P_x,y)) \therefore (\forall y)((\exists x)(P_x,y))
