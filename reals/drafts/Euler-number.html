The Exponential Function

For all real x, the function \exp(x)\colon\mathbb{R}\mapsto\mathbb{R} is defined by the functional limit:
\exp(x)=\lim\left(1+\frac{x}{n}\right)^n

Properties of the Exponential Function

Definition: The number e, called Euler's Number, is a constant defined as the exponential function evaluated at the real value 1.
e = \exp(1) = \sum_{n=0}^{\infty}\frac{1}{n!}

Euler's formula derivation: for any real number x,
\exp(\imath x) = \cos(x) + \imath\sin(x)
proof:
\begin{align*} \exp(\imath x) &= \sum_{n=0}^{\infty}\frac{\left(\imath x\right)^n}{n!}\\ &= \frac{\left(\imath x\right)^0}{0!} + \frac{\left(\imath x\right)^1}{1!} + \frac{\left(\imath x\right)^2}{2!} + \frac{\left(\imath x\right)^3}{3!} + \frac{\left(\imath x\right)^4}{4!} + \frac{\left(\imath x\right)^5}{5!} + \frac{\left(\imath x\right)^6}{6!} + \cdots \\ &= 1+\imath x-\frac{x^2}{2!}-\frac{\imath x^3}{3!}+\frac{x^4}{4!}+\frac{\imath x^5}{5!}-\frac{x^6}{6!} + \cdots\\ &= 1-\frac{x^2}{2!}+\frac{x^4}{4!}-\frac{x^6}{6!}+\cdots + \imath\left(x-\frac{x^3}{3!}+\frac{x^5}{5!}+\cdots\right)\\ &= \sum_{n=0}^{\infty}\frac{\left(-1\right)^n x^{2n}}{\left(2n\right)!} + \imath\sum_{n=0}^{\infty}\frac{\left(-1\right)^n x^{2n+1}}{\left(2n+1\right)!}\\ &= \cos(x) + \imath\sin(x)\\ \end{align*}
for any complex number z = \alpha + \beta\imath
\alpha + \beta\imath = r\exp\left(\imath\theta\right)
where r = \sqrt{\alpha^2+\beta^2} and \theta = \arctan{\frac{\beta}{\alpha}} proof:

Let \theta = \arctan{\frac{\beta}{\alpha}}. Then

\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{\beta}{\alpha}
That is, \beta = k_1\sin\theta and \alpha = k_2\cos\theta. Then
\begin{align*} \alpha + \beta\imath &= k_2\cos\theta + \imath k_1\sin\theta\\ &= k_2\left(\cos\theta\right) + \imath k_1\left(\sin\theta\right)\\ &= r\left(\cos\theta + \imath\sin\theta\right)\\ &= r\exp\left(\imath\theta\right) \end{align*}
