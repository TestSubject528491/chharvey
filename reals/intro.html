Propositional Logic

(The Axiom of Existence) There exists a set with no elements, and that set is unique.

This set is denoted and is called the empty set.

Preliminary Definitions

A set is that which may contain elements. These elements, in turn, are also sets.

If a set \(X\) contains a set \(x\), we state, \(x\in X\) (\(x\) is an element of \(X\)), or alternatively, \(X\owns x\) (\(X\) contains \(x\)). If a set \(X\) does not contain a set \(x\), we state, \(x\not\in X\) or, \(X\not\owns x\).

It is interesting that the first node of this axiomatic system is a cyclic definition. All systems must start from one point, and the idea of a set is that starting point. If we take for granted the definition of a set, we can use it to construct more building blocks in the system.

A proposition \(\mathsf{p}\) is a statement concerning sets. This statement may be \(\text{true}\) or \(\text{false}\).

When we assert that \(\mathsf{p}\) is \(\text{true}\), we state, \(\mathsf{p}\).

Given a proposition \(\mathsf{p}\), the assertion \(\mathsf{p}\) is \(\text{false}\) is itself a proposition, called the negation \(\lnot\mathsf{p}\) of \(\mathsf{p}\).

When we assert that \(\mathsf{p}\) is \(\text{false}\), we state, not \(\mathsf{p}\).

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the assertion if \(\mathsf{p}\) is \(\text{true}\) then \(\mathsf{q}\) must be \(\text{true}\) is itself a proposition, called a (material) implication \(\mathsf{p}\implies\mathsf{q}\).

There are a number of other ways to express a material implication.
When \(\mathsf{p}\implies\mathsf{q}\) is \(\text{true}\), we state,
\(\mathsf{p}\) is the antecedent \(\mathsf{q}\) is the consequent
\(\mathsf{p}\) is the hypothesis \(\mathsf{q}\) is the conclusion
\(\mathsf{p}\) only if \(\mathsf{q}\) \(\mathsf{q}\) if \(\mathsf{p}\)
\(\mathsf{p}\) implies \(\mathsf{q}\) \(\mathsf{q}\) is implied by \(\mathsf{p}\)
\(\mathsf{p}\) is sufficient for \(\mathsf{q}\) \(\mathsf{q}\) is necessary for \(\mathsf{p}\)
\(\mathsf{p}\) is stronger than \(\mathsf{q}\) \(\mathsf{q}\) is weaker than \(\mathsf{p}\)

Given an implication \(\mathsf{r}\) defined by \(\mathsf{p}\implies\mathsf{q}\),

  • the implication \(\mathsf{q}\implies\mathsf{p}\) is called the converse of \(\mathsf{r}\).
  • the implication \(\lnot\mathsf{p}\implies\lnot\mathsf{q}\) is called the inverse of \(\mathsf{r}\).
  • the implication \(\lnot\mathsf{q}\implies\lnot\mathsf{p}\) is called the contrapositive of \(\mathsf{r}\).

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the disjunction \(\mathsf{p}\lor\mathsf{q}\) is a proposition defined by the implication \(\lnot\mathsf{p}\implies\mathsf{q}\).

When \(\mathsf{p}\lor\mathsf{q}\) is \(\text{true}\), we state, \(\mathsf{p}\) or \(\mathsf{q}\).

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the conjunction \(\mathsf{p}\land\mathsf{q}\) is a proposition defined by the negation \(\lnot(\mathsf{p}\implies\lnot\mathsf{q})\).

When \(\mathsf{p}\land\mathsf{q}\) is \(\text{true}\), we state, \(\mathsf{p}\) and \(\mathsf{q}\).

A discussion on or and and.

The disjunction \(\mathsf{p}\lor\mathsf{q}\) is \(\text{true}\) exactly when the assertion that “if \(\mathsf{p}\) is \(\text{false}\) then \(\mathsf{q}\) must be \(\text{true}\)” is \(\text{true}\).

The conjunction \(\mathsf{p}\land\mathsf{q}\) is \(\text{true}\) exactly when the assertion that “if \(\mathsf{p}\) is \(\text{true}\) then \(\mathsf{q}\) must be \(\text{false}\)” is \(\text{false}\).

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the (material) equivalence \(\mathsf{p}\iff\mathsf{q}\) is a proposition defined by the conjunction \((\mathsf{p}\implies\mathsf{q})\land(\mathsf{q}\implies\mathsf{p})\).

\[(\mathsf{p}\iff\mathsf{q}) \equiv (\mathsf{p}\implies\mathsf{q})\land(\mathsf{q}\implies\mathsf{p})\]

There are a number of other ways to express a material equivalence.
When \(\mathsf{p}\iff\mathsf{q}\) is \(\text{true}\), we state,
\(\mathsf{p}\) if and only if (iff) \(\mathsf{q}\)
\(\mathsf{p}\) implies and is implied by \(\mathsf{q}\)
\(\mathsf{p}\) is necessary and sufficient for \(\mathsf{q}\)
\(\mathsf{p}\) is materially equivalent to \(\mathsf{q}\)

Given a proposition \(\mathsf{p}\), the tautology \(\top\) is a proposition defined by the implication \(\mathsf{p}\implies\mathsf{p}\).

Given a proposition \(\mathsf{p}\), the contradiction \(\bot\) is a proposition defined by the negation \(\lnot(\mathsf{p}\implies\mathsf{p})\).

In summary:
\(\mathsf{p}\) \(\mathsf{q}\) \(\lnot\mathsf{p}\) \(\lnot\mathsf{q}\) \(\mathsf{p}\implies\mathsf{q}\) \(\mathsf{p}\lor\mathsf{q}\) \(\mathsf{p}\land\mathsf{q}\) \(\mathsf{p}\iff\mathsf{q}\) \(\top\) \(\bot\)

A proposition exists.

can be thought of as the Bootstrap Axiom, because it is a proposition that asserts the existence of itself. By merely existing it is fulfilling its prophecy. Simply put, it proves itself true. Think for a moment about what would happen if it were false. Then there would be no propositions, and it simply wouldn't exist. Thus at this point we can conclude that there is at least one true proposition.

(The Principle of Bivalence) It is always the case that a proposition is \(\text{true}\) or \(\text{false}\), and it is never the case that a proposition is both \(\text{true}\) and \(\text{false}\).

The logical values \(\text{true}\) and \(\text{false}\) are both collectively exhaustive and mutually exclusive. In other words, a proposition must be either true or false, and it can only be one of these.

The empty set is a subset of every set.

\[\forall X (\varnothing \subseteq X)\]

Since the empty set has no elements, \(t\in\varnothing \implies t\in X\) by trivial implication. By definition of subset, \(\varnothing \subseteq X\).

demorgan's laws:

\(\mathsf{p}\lor\mathsf{q}\) is false iff \(\mathsf{p}\) is false and \(\mathsf{q}\) is false.

\(\mathsf{p}\land\mathsf{q}\) is false iff \(\mathsf{p}\) is false, or \(\mathsf{q}\) is false, or both are false.

If \(\lnot\mathsf{p}\) is the assertion that \(\mathsf{p}\) is false, then the assertion that \(\lnot\mathsf{p}\) is false, \(\lnot\lnot\mathsf{p}\), namely, the assertion that the assertion that \(\mathsf{p}\) is false is false, is the assertion that \(\mathsf{p}\) is true. In other words,
