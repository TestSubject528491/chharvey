Lesson Implementation

Warm-up

Hand students the Algebra Code worksheet. Let them work on it independently. They should be done very quickly.

Instruction

Worksheets

Whole Number – One Variable

Whole Number – Two Variables

Practice Problems

  1. Evaluate \(ab-c\) when \(a=2\), \(b=3\), and \(c=4\).
    1. \(2\)
    2. \(3\)
    3. \(4\)
    4. \(5\)
  2. When \(a=2\), \(b=3\), and \(c=4\), choose the expression with a value of \(25\).
    1. \(a+b-c+5\)
    2. \(7c-2a\)
    3. \(2bc+1\)
    4. \(2b-4\)
  3. Look at the expression below:
    \[2b+3a\]
    Which set of variables will make the expression equal to \(12\)?
    1. \(a=1\), \(b=2\)
    2. \(a=2\), \(b=3\)
    3. \(a=3\), \(b=4\)
    4. \(a=3\), \(b=2\)

After reviewing the problems, have students summarize.
