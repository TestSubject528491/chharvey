Unit 2 Project: Function Painting

Created October 12, 2012.

Goal

To provide a tool for students to demonstrate their understanding of properties, families, and transformations of functions, while showcasing their artistic ability and creativity.

Task

Given a set of 10 function families, the student will construct a painting that includes an instance of each parent function listed. The instance functions the student uses may be any variation or transformation of the parent functions required. Accompanying the painting, the student will submit a list of the instance functions used, and properties of those functions.

Grading Rubric

Required Functions

  • linear
  • quadratic
  • cubic
  • rational
  • square root
  • cube root
  • exponential growth or decay
  • logarithmic growth or decay
  • absolute value
  • miscellaneous (your choice)

Mathematical Accuracy

The Cartesian x-y-plane will be shown, with accurate labels for axes and tick marks.

There will be at least ten functions drawn. At least one function from each required type will be graphed properly. The parent function may be used. There may be more than one function from each family drawn.

The graph of each function and its following properties will be accurately depicted:

  • proper intercept points
  • proper turning points
  • proper curvature and slope

The student may choose to restrict the domain of the functions he or she uses. If this is the case, the student will place open or closed endpoints on the painting accordingly.

The student will submit a list of functions used in the painting. This list will be error free and will include

  1. the function's equation
  2. the function's family name
  3. a description of the transformation, if any, in English
  4. the (possibly restricted) domain and range of the function, using set-builder notation
  5. the x- and y-intercepts of the function, if any
  6. the turning points of the function, if any
  7. the end behavior of the function, if any

The functions on the list will correspond to the functions on the graph in some way. The student may choose to color-code, number, letter, footnote, etc. the functions. The audience should be able to look at a function and then find it on the list, without knowing anything about functions.

Artistic Requirements and Guidelines

The painting can represent anything concrete or abstract, such as an object, idea, action, or emotion. The object of the painting must be somewhat recognizable by the audience. The painting must have some aesthetic appeal.

In addition to the color white, the painting must display at least two other colors, one of which may be black.

The painting can be done on, and done with, any medium. The size of the painting must be a minimum of 8.5in by 11in, or equivalent area.
