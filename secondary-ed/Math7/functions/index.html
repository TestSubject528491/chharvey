Unit 3: Functions

Updated Nov 24, 2013.

This unit explores relations between two variables and patterns of integers and rational numbers. Students will identify and derive functions of rational numbers using a variety of representations. They will also analyze and generalize sequences of integers and rational numbers.

Learning Objectives

Relations

  • Graph individual points in the Cartesian plane.
    • given an ordered pair, plot the point in the plane
    • given a point in the plane, determine its x- and y-coordinates
    • identify Quadrants in the Cartesian plane
  • 7.12 Represent relations with tables, graphs, rules, and words.
    1. Represent finite relations in set form (as a set of points), table form, and graph form.
      1. convert between a set of points and a table
      2. convert between a set of points and a graph
      3. convert between a table and a graph
    2. Distinguish whether a relation is a function, given in set form, table form, or graph form. Relations given in graph form may be infinite.
    3. Given a linear function in equation form (\(y=\frac{p}{q}x+b\) where \(p\), \(q\), and \(b\) are integers):
      1. create a table of representative x-and-y pairs
      2. create a set of representative points
      3. sketch the graph on a point-by-point basis
    4. Given a linear function in representative set form, representative table form, or graph form, write an equation that describes the function.
    5. Describe linear functions using verbal statements.

Sequences

  • 7.2 Determine patterns in sequences of objects (numbers, shapes, etc.).
    • Patterns will be recursive, that is, the next term in the sequence will be determined by its predecessor. Recursive patterns will be represented by variable expressions.
    • Given an arithmetic sequence, find the common difference.
    • Given a geometric sequence, find the common ratio.
    • Create a sequence given a variable expression describing the pattern.
