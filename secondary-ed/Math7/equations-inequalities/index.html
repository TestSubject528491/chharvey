Learning Objectives

Equations

  • 7.14a
    • Solve one-variable linear equations in one step. Where \(a\), \(b\), \(c\), \(p\), and \(q\) are integers, equations of the following forms will be given:
      • \(x+b=c\)
      • \(ax=c\)
      • \(\frac{x}{q}=c\)
      • \(\frac{p}{q}x=c\)
    • Solve one-variable linear equations in two steps. Where \(a\), \(b\), \(c\), \(p\), and \(q\) are integers, equations of the following forms will be given:
      • \(ax+b=c\)
      • \(\frac{x}{q}+b=c\)
      • \(\frac{p}{q}x+b=c\)
      • \(p\left(x+b\right)=c\)
      • \(\frac{x+b}{q}=c\)
  • 7.14b Solve practical problems requiring the solution of one-variable linear equations.
    • Given a word problem, students will select or construct an appropriate equation that models the real-world situation.
    • Given a word problem, students will find the solution using a two-step linear equation.

Inequalities

  • 7.15a Solve one-variable linear inequalities in one step.
  • 7.15b Graph solutions to one-variable linear inequalities on a number line.
