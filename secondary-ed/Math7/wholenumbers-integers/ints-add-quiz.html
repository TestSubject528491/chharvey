Math 7 — Mr. Harvey

Quiz 1.3: Integer Addition

Use a No. 2 Pencil. Show all work. No calculator!

#1–3: Draw a representation of each sum, using the specified model.

  1. Show that \(-2+5=3\), using a Chip Model.

  2. Show that \(-5+1=-4\), using a Block Model.

  3. Show the sum of \(3+{^-}7\), using a Number Line Model.

#4–6: Write a sum that is represented by each of the following diagrams. Then evaluate the sum.

#7–10: Evaluate the following sums. Show all work.

  1. \(-15+8\)
  2. \(-23+79\)
  3. \(-75+{^-}34\)
  4. \(56+{^-}56\)
