Quiz 1.4: Integer Subtraction

Use a No. 2 Pencil. Show all work. No calculator!

#1–4: Draw a representation of each difference, using the specified model.

  1. Show that \(6-{^-}8=14\), using a Chip Model.

  2. Show the difference of \(-9-{^-}6\), using a Chip Model.

  3. Show that \(-5-3=-8\), using a Block Model.

  4. Show the difference of \(10-15\), using a Block Model.

#5–6: Write a difference that is represented by each of the following diagrams. Then evaluate the difference.

#7–10: Evaluate the following differences. Show all work.

  1. \(-9-{^-}1\)
  2. \(-4-{^-}5\)
  3. \(-9-5\)
  4. \(-4-9\)
