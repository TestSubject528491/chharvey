Lesson 1.1: Integer Addition

SOL
7.3a The student will model addition, subtraction, multiplication, and division of integers.
7.3b The student will add, subtract, multiply, and divide integers.
Timeframe
Week I.3
Time required
3 days
Lesson created

Lesson updated

Preparation

Objectives

  • The student will model addition of integers (with the same sign and different signs) using two-colored counters, a two-sided mat, and a number line.
  • The student will evaluate the sum of two integers, with the same sign or different signs.

Essential Questions

  • What is an integer?
  • How do I add integers when they have the same sign?
  • How do I add integers when they have different signs?

Vocabulary

zero pair
a pair of units that, when added, make 0
sum
the answer to an addition problem

Lesson Implementation

Warm-up

Instruction

# Adding Integers (Page 15) #

## Chips / Blocks ##
1. Put on 1st number
2. Put on 2nd number
3. Cancel zero pairs

## Number Line ##
1. Start at zero
2. Walk forward to 1st number
3. Face direction of 2nd number
4. Walk forward that many steps

# Subtracting Integers (Page 17) (Lesson 1.4) #

## Chips / Blocks ##
1. Put on 1st number
2. Add the necessary number of zero pairs
3. Take away 2nd number

## Number Line ##
1. Start at zero
2. Walk forward to 1st number
3. Face direction of 2nd number
4. Walk backward that many steps
