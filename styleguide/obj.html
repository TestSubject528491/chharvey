The Block Object

The Inline Object

A simple inline object. Use this class to display an element as an inline element, with zero line-height to maintain vertical rhythm.

This class is useful for creating inline lists, that is, lists whose items are each displayed one right after another, inline.

The Box Object

Use the Box object for stand-alone, highlighted blocks. They are useful for things like alerts, pop-ups, and asides. Since they have a border radius, they don’t look good in tables or table-like structures. Add class .o-Block if necessary.

The Textbox Object

The Textbox Object is similar to the Box object except that it is used for inline text only, such as labels or badges. It is usually instantiated by a small element, but that depends on context. Add class .o-Inline if necessary.

The Bare List

Use the Bare List object to remove the usual padding and list item markers accompanied by (un)ordered lists. The result is a list of items, each on its own line. Combine this class with other classes to create a variety of styles.

The Flex Grid

A Flex Grid is simply a list of items arranged like blocks in a grid. It has the structure of a list in that all items share the same container (unlike tables where items are in different rows), while at the same time appearing like a table with rows and columns.

The Flex Grid can be flexible and responsive, adjusting the number of columns based on screen size. By default, each item has full width. To achieve a grid-like appearance, set the widths of each item as necessary.

The container must have the .o-GridFlex class (and, if necessary, class .o-ListBare to remove normal list styles). Each item must have the .o-GridFlex__Item class (and, if necessary, class .o-Block).

See my article The GridFlex Object.

The Golden Container

See my article griddemo.html for docs on the GoldenContainer object.

The following is a .o-GoldenContainer Object, containing 3 children:

  1. a .o-GoldenContainer__Content Object
  2. a .o-GoldenContainer__Sidebar.o-GoldenContainer__Sidebar--major Object
  3. a .o-GoldenContainer__Sidebar.o-GoldenContainer__Sidebar--minor Object

Sometimes you may have objects inside the Content that you wish to make appear as a sidebar or as if it weren’t in the Content.

The following .o-GoldenContainer Object contains only a .o-GoldenContainer__Content Object, which contains 3 children:

  1. a .o-GoldenContainer__Content__Page Object
  2. a .o-GoldenContainer__Content__SideMajor Object
  3. a .o-GoldenContainer__Content__SideMinor Object

Even though these are children of .o-GoldenContainer__Content, they appear as if they are outside (for example, inside a __Sidebar--major or __Sidebar--minor).
