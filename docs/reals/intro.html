Section 1.A: Logic

A proposition \(\mathsf{p}\) is \(\text{true}\) or \(\text{false}\), and may or may not be determined or dependent.

The proposition \(\mathsf{p}\) may also be referred to as a logical expression, or a material statement. To make the assertion that \(\mathsf{p}\) is \(\text{true}\), we simply state \(\mathsf{p}\).

A proposition exists.

can be thought of as the Bootstrap Axiom, because it is a proposition that asserts the existence of itself. Indeed, there were no propositions formally stated prior to this one, so by merely existing it is fulfilling its prophecy. Simply put, it proves itself true. This proposition makes a great choice for the first axiom, because we can use it to construct other propositions. At this point we can conclude that there is at least one true proposition. Given a proposition \(\mathsf{p}\), the assertion that \(\mathsf{p}\) is \(\text{false}\) is itself a proposition, called the negation of \(\mathsf{p}\), denoted \(\lnot\mathsf{p}\).

To assert that \(\mathsf{p}\) is false is to assert that \(\lnot\mathsf{p}\) is true, and we state, \(\lnot\mathsf{p}\).

Given a proposition \(\mathsf{p}\) and a proposition \(\mathsf{q}\), the (material) implication \(\mathsf{p}\implies\mathsf{q}\) is a proposition that is true exactly when the assertion that \(\mathsf{p}\) is true requires that \(\mathsf{q}\) be true.

When \(\mathsf{p}\implies\mathsf{q}\) is true, we say, \(\mathsf{p}\) implies \(\mathsf{q}\), or, \(\mathsf{q}\) is implied by \(\mathsf{p}\).

There are a number of other ways to express a material implication.
\(\mathsf{p}\implies\mathsf{q}\)
\(\mathsf{p}\) implies \(\mathsf{q}\) \(\mathsf{q}\) is implied by \(\mathsf{p}\)
\(\mathsf{p}\) only if \(\mathsf{q}\) \(\mathsf{q}\) if \(\mathsf{p}\)
\(\mathsf{p}\) is sufficient for \(\mathsf{q}\) \(\mathsf{q}\) is necessary for \(\mathsf{p}\)
\(\mathsf{p}\) is the antecedent \(\mathsf{q}\) is the consequent
\(\mathsf{p}\) is the hypothesis \(\mathsf{q}\) is the conclusion

Given an implication \(\mathsf{r}\) which is defined by \(\mathsf{p}\implies\mathsf{q}\),

  • the implication \(\mathsf{q}\implies\mathsf{p}\) is called the converse of \(\mathsf{r}\).
  • the implication \(\lnot\mathsf{p}\implies\lnot\mathsf{q}\) is called the inverse of \(\mathsf{r}\).
  • the implication \(\lnot\mathsf{q}\implies\lnot\mathsf{p}\) is called the contrapositive of \(\mathsf{r}\).

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the disjunction \(\mathsf{p}\lor\mathsf{q}\) is a proposition that is true exactly when the implication \(\lnot\mathsf{p}\implies\mathsf{q}\) is true.

That is, when the assertion that \(\mathsf{p}\) is false requires that \(\mathsf{q}\) be true, the disjunction \(\mathsf{p}\lor\mathsf{q}\) (read, \(\mathsf{p}\) or \(\mathsf{q}\)) is true.

Informally, \(\mathsf{p}\lor\mathsf{q}\) is true iff \(\mathsf{p}\) or \(\mathsf{q}\) (or possibly both) are true.

Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the conjunction \(\mathsf{p}\land\mathsf{q}\) is a proposition that is true exactly when the implication \(\mathsf{p}\implies\lnot\mathsf{q}\) is false.

That is, the conjunction \(\mathsf{p}\land\mathsf{q}\) (read, \(\mathsf{p}\) and \(\mathsf{q}\)) is true exactly when it is not the case that the assertion that \(\mathsf{p}\) is true requires that \(\mathsf{q}\) be false.

Informally, \(\mathsf{p}\land\mathsf{q}\) is true iff \(\mathsf{p}\) and \(\mathsf{q}\) are both true.

Given a proposition \(\mathsf{p}\), a tautology \(\top\) is the proposition \(\mathsf{p}\lor\lnot\mathsf{p}\).

Given a proposition \(\mathsf{p}\), a contradiction \(\bot\) is the proposition \(\mathsf{p}\land\lnot\mathsf{p}\).

(The Classic Laws of Thought) Given a proposition \(\mathsf{p}\),

  • (The Law of Excluded Middle) It is always the case that \(\mathsf{p}\) is true or false.

    \[\mathsf{p} \lor \lnot\mathsf{p}\]

  • (The Law of Non-Contradiction) It is never the case that \(\mathsf{p}\) is both true and false.

    \[\lnot(\mathsf{p} \land \lnot\mathsf{p})\]

The tautology \(\top\) is always \(\text{true}\), and the contradiction \(\bot\) is always \(\text{false}\).

The logical values \(\text{true}\) and \(\text{false}\) are both collectively exhaustive and mutually exclusive. In other words, a proposition must be either true or false, and it can only be one of these.

Axiom 1.1.1-7 (The Axiom of Double Negation) \(\mathsf{p}\) is true if and only if \(\lnot\mathsf{p}\) is false.

\[\mathsf{p} \equiv \lnot\lnot\mathsf{p}\]

Informally, we know the following propositions are equivalent:

  • \(\mathsf{p}\land\mathsf{q}\) is true
  • \(\lnot(\lnot\mathsf{p} \lor \lnot\mathsf{q})\) is true
  • \(\lnot\mathsf{p} \lor \lnot\mathsf{q}\) is false
  • \(\lnot\mathsf{p}\) is false and \(\lnot\mathsf{q}\) is false
  • \(\mathsf{p}\) is true and \(\mathsf{q}\) is true

We cannot, however, state this formally, because jumping from \(\lnot\mathsf{p}\) is false to \(\mathsf{p}\) is true in the list above requires a very important axiom, which cannot be presented until we introduce conditional statements.

Section 1.B: Metalogic

Definition Given propositions \(\mathsf{p}\) and \(\mathsf{p}\), the logical implication \(\mathsf{p}\therefore\mathsf{q}\) is true iff the proposition \(\mathsf{p}\implies\mathsf{q}\) is a tautology \(\top\).

When \(\mathsf{p}\therefore\mathsf{q}\) we say \(\mathsf{p}\) logically implies \(\mathsf{q}\) or \(\mathsf{p}\), therefore \(\mathsf{q}\) or \(\mathsf{q}\), because \(\mathsf{p}\) (\(\mathsf{q}\because\mathsf{p}\)).

Definition Given propositions \(\mathsf{p}\) and \(\mathsf{q}\), the logical equivalence \(\mathsf{p}\equiv\mathsf{q}\) is true iff the proposition \((\mathsf{p}\therefore\mathsf{q})\land(\mathsf{q}\therefore\mathsf{p})\) is true.
