Class NormalDistribution

A normal (Gaussian) distribution with a specified mean and standard deviation. If the mean and/or standard deviation are not given, they default to 0 and 1 respectively.

Constructor

new NormalDistribution(m, s)
Param m
optional number defaults to 0; the statistical center of the random variable
Param s
optional number defaults to 1; the statistical spread of the random variable

Fields

Instance Variables

the statistical center of the random variable. immutable

the statistical spread of the random variable. immutable

this.area(a, b, n)
Description
Returns the area under the PDF in the closed interval [a, b]. The following approximations hold:
  • [-1s + m, 1s + m]: about 0.6827
  • [-2s + m, 2s + m]: about 0.9545
  • [-3s + m, 3s + m]: about 0.9973
For large values of a or b (larger than ±10), this method will return an inaccurate result. Specify a larger value of n to compensate.
Param a
required number; the minimum of the interval
Param b
required number; the maximum of the interval
Param n
optional integer 0 \lt n \le 150 defaults to 100; the accuracy of this method

this.rand()
Description
Returns a random number within the open interval (-\infty, \infty) and with this distribution’s mean and standard deviation. Note: while the range of this method’s output is technically all of \RR, the following probabilities hold:
  • the output will be in (-1s + m, 1s + m): about 68.27%
  • the output will be in (-2s + m, 2s + m): about 95.45%
  • the output will be in (-3s + m, 3s + m): about 99.73%
