Color Schemes

Main Color Scheme

Aperture White

The purity of Aperture Science in all its glory. This dazzling brilliance of bluish light seen on the portalable panels coated with Conversion Gel makes the background emit an eerie glow, like the full moon.

  • body background

Aperture Black

The polar opposite of Aperture White. Representing the true dark nature of the Enrichment Center, this is the color of the panels that inhibit portals. A hint of orange is infinitesimally noticeable. What better way to complement the background color than the color of normal text?

  • body text

Lake Superior

This Great Lake borders Ontonagon, MI, the location of the Aperture Science Enrichment Center. A perfect fit for superior headings on a page.

  • major heading text
  • drop caps

Facility Abyss

This color can be seen in the endless depths of the subterranean Enrichment Center. This color is for minor headings of a page.

  • minor heading text

Atlas

Short and stout, Atlas is one of the two robots used for testing. Untouched hyperlinks are dedicated to his digital iris.

  • link text
  • breadcrumb text

P-Body

Tall and wiry, P-body is one of the two robots used for testing. Hovered hyperlinks are dedicated to his (or her?) digital iris.

  • hover link text
  • hover breadcrumb text

Neurotoxin

Long ago an accident resulted in the unleashing of this deadly vapor and the murdering of the entire human staff of Aperture Laboratories. Some say it could happen again, but this time on the test subjects…

  • caption, postal, figure, block quote border
  • table border
  • footnote border

Deploying

The one word you never want to hear without a portal gun. Just as the iris of a Sentry Turret, this color is briefly seen, providing a subtle accent on the site.

  • subheading text
  • active link text
  • u text
  • mark background
  • help outline
This color scheme is based on the Portal video game series. See the portal color scheme page for more fun colors.

Code Color Scheme

The following colors are used for syntax highlighting of displayed code.

Solarized Base03

  • dark code background

Solarized Base3

  • light code background

Solarized Base02

  • dark code border

Solarized Base2

  • light code border

Solarized Base01

  • light code emphasis
  • dark code comment

Solarized Base1

  • light code comment
  • dark code emphasis

Solarized Base00

  • light code text

Solarized Base0

  • dark code text

Solarized Red

  • programming control (if, for, switch, return)

Solarized Cyan

  • HTML attribute values
  • strings

Solarized Orange

  • delimiters and operators (+, ==, ., new, this)

Solarized Blue

  • HTML tags
  • CSS selectors
  • Less/JS/Latex function and class names

Solarized Yellow

  • numbers, colors, boolean constants, characters, entities
  • CSS property values

Solarized Violet

  • variable names
  • Less/JS/Latex command arguments

Solarized Green

  • HTML attribute names
  • CSS property names
  • pre-defined keywords (var, function)

Solarized Magenta

  • string delimiters (" and ')

Message Color Scheme

Each of the five message colors are equidistant in hue. The border colors are the base hues, the text colors are a shade, and the background colors are a tint. Both the shade and the tint form a Golden proportion with the base hue.

warning text warning border warning background
caution text caution border caution background
success text success border success background
info text info border info background
help text help border help background
