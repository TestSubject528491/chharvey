Font Scheme

The base font-family is used for the main body copy. It is a sans-serif polyweight type with a moderate thick/thin transition. Being a sans-serif, its italic is actually an oblique, which just slants the glyphs. Suggested typefaces include Optima, Lucida Grande, Tahoma.

The prose font-family is used for text in formal documents and articles. It is a serif oldstyle type, which is easy on the eyes and can be read at extensive length. Like the base font, it, too, has a moderate thick/thin transition, but has diagonal stress and serifs, and true italics, which means it uses a whole new set of glyphs. Suggested typefaces include Baskerville, Palatino, Cochin.

The heading font-family is used only for headings of body copy (not table headings, etc.). It is a sans-serif type with little to no thick/thin transition. It is extremely lightweight, which makes a great heading when boldened, but it should not be used for extensive amounts of text. Suggested typefaces include Avenir, Seravek.

The accent font-family is used for labels, block quotes, butons/keys, and other special elements. Like the heading font-family, it is a monoweight sans-serif, yet slightly heavier. Suggested typefaces include Geneva, Tahoma.

The code font-family is a monospace type used for… well, code. Personally, I dislike monospace fonts with serifs, so this family is sans-serif. Suggested typefaces include Menlo, Monaco, Andale Mono.

The math font-family is a serif modern type use for mathematical spans and spans in foreign languages. It has a radical thick/thin transition, vertical stress, and straight horizontal serifs. With a cold yet elegant look, this font family makes long spans of text hard to read, but that’s okay because this font is used only for short expressions. Suggested typefaces include Didot, Bodoni 72, Times.

The script font-family is used for very fancy elements such as drop-caps and subheadings. Use this font family sparingly. Suggested typefaces include Zapfino.

Snell Roundhand is another script typeface used sparingly throughout the site.

Portal Color Scheme

This color scheme is based on the Portal video game series.

Main Portal Colors

Aperture White

The purity of Aperture Science in all its glory. This dazzling brilliance of bluish light seen on the portalable panels coated with Conversion Gel makes the background emit an eerie glow, like the full moon.

  • body background

Aperture Black

The polar opposite of Aperture White. Representing the true dark nature of the Enrichment Center, this is the color of the panels that inhibit portals. A hint of orange is infinitesimally noticeable. What better way to complement the background color than the color of normal text?

  • body text

Lake Superior

This Great Lake borders Ontonagon, MI, the location of the Aperture Science Enrichment Center. A perfect fit for superior headings on a page.

  • major heading text
  • drop caps

Facility Abyss

This color can be seen in the endless depths of the subterranean Enrichment Center. This color is for minor headings of a page.

  • minor heading text
  • table background (tints)

Atlas

Short and stout, Atlas is one of the two robots used for testing. Untouched hyperlinks are dedicated to his digital iris.

  • link text
  • breadcrumb text

P-Body

Tall and wiry, P-body is one of the two robots used for testing. Hovered hyperlinks are dedicated to his (or her?) digital iris.

  • hover link text
  • hover breadcrumb text

Neurotoxin

Long ago an accident resulted in the unleashing of this deadly vapor and the murdering of the entire human staff of Aperture Laboratories. Some say it could happen again, but this time on the test subjects…

  • caption, postal, figure, block quote border
  • table border
  • footnote border

Deploying

The one word you never want to hear without a portal gun. Just as the iris of a Sentry Turret, this color is briefly seen, providing a subtle accent on the site.

  • subheading text
  • active link text
  • u text
  • s text
  • mark background
  • help outline

Message Color Scheme

Each of the five message colors are equidistant in hue. The border colors are the base hues, the text colors are a shade, and the background colors are a tint. Both the shade and the tint form a Golden proportion with the base hue.
